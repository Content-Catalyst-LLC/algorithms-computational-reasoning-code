# Use Boundaries

This companion folder is educational and interpretive. It is not a complete historical database, not a philological edition, not a manuscript catalogue, and not proof of any single-person origin claim.

Use these files to support:

- teaching;
- conceptual mapping;
- historical framing;
- algorithmic-reasoning interpretation;
- structured comparison of procedural themes;
- computational scaffolds for humanities-oriented analysis.

Do not use them to claim that al-Khwārizmī invented all algorithms or that medieval procedures were identical to modern software.

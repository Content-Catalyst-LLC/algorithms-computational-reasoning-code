# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `place_value_calculator.py` — compute digit × base^position.
- `algebra_balance_calculator.py` — apply the same addition to both sides of an equation.
- `equation_type_classifier.py` — classify a toy polynomial degree as linear, quadratic, or higher-order.
- `table_interpolation_calculator.py` — linear interpolation between two table values.
- `procedural_legacy_score_calculator.py` — average procedure, representation, transmission, application, and modern resonance.
- `transmission_weight_calculator.py` — weighted score for method movement through languages and teaching settings.

## R calculators

- `procedural_legacy_score.R` — compute an interpretive legacy score.

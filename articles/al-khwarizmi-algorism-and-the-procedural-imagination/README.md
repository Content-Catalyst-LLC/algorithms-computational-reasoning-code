# Al-Khwārizmī, Algorism, and the Procedural Imagination

This companion folder supports the Sustainable Catalyst article **Al-Khwārizmī, Algorism, and the Procedural Imagination**.

It provides reproducible, teaching-oriented workflows for mapping al-Khwārizmī's procedural legacy: algorism, written arithmetic, Hindu-Arabic numerals, place-value notation, algebraic restoration and balancing, problem classification, verbal procedures before symbolic algebra, astronomical tables, Latin reception, etymology, transmission, and the procedural imagination.

## What this folder demonstrates

- how al-Khwārizmī's legacy can be studied as procedural, representational, transmissional, practical, and modern-resonant history;
- how to distinguish al-Khwārizmī's centrality to algorithmic naming and algorism from an inaccurate single-origin claim;
- how written arithmetic, place value, algebraic transformation, problem classification, verbal procedure, astronomical tables, and Latin reception can be mapped as related but distinct themes;
- how calculators can demonstrate place value, algebraic balancing, equation-type classification, tabular interpolation, procedural legacy scoring, and transmission weighting;
- how documentation can support accurate, appreciative, non-reductive historical interpretation.

## Repository structure

- `python/` — reference procedural legacy map workflow and tests
- `r/` — procedural legacy diagnostics and figures
- `sql/` — schema and queries for legacy themes, scores, and interpretation cautions
- `calculators/` — self-contained place-value, balancing, classification, interpolation, legacy-score, and transmission-weight calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, etymology notes, procedural imagination notes, origin-story cautions, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and interpretive. They are educational scaffolds, not claims of exhaustive historical proof.

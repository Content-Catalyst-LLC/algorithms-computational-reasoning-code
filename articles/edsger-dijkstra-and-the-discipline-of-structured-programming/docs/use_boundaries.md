# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching structured programming;
- explaining program correctness and invariants;
- illustrating weakest-precondition reasoning;
- discussing guarded commands and formal derivation;
- exploring Dijkstra's shortest-path algorithm;
- reviewing AI-generated code through structured reasoning.

Do not treat synthetic scores as formal proofs, safety certifications, production verification, or security guarantees.

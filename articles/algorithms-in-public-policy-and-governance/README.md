# Algorithms in Public Policy and Governance

This companion folder supports the Sustainable Catalyst article **Algorithms in Public Policy and Governance**.

It provides reproducible, teaching-oriented workflows for public-sector algorithmic systems: automated eligibility, risk scoring, triage, fraud detection, regulatory targeting, public-service delivery, administrative accountability, due process, transparency, appeals, procurement, vendor accountability, public-sector AI governance, monitoring, audit trails, and institutional responsibility.

- Previous: `algorithms-in-knowledge-architecture`

## What this folder demonstrates

- how public algorithmic use cases can be reviewed for rights impact, due process, transparency, human review, appeal readiness, data quality, vendor accountability, monitoring, and public value;
- how procedural readiness and governance readiness can support deployment decisions;
- how high-rights-impact systems should be stopped, redesigned, or independently overseen when due process is weak;
- how public algorithmic governance requires inventories, legal authority review, impact assessment, vendor accountability, monitoring, and stop rules;
- how calculators, tests, SQL queries, Canvas cards, and multilingual examples can support accountable public-sector algorithmic governance.

## Repository structure

- `python/` — reference public governance audit workflow and tests
- `r/` — diagnostic summaries and figures
- `sql/` — schema and queries for public algorithm use cases, governance readiness, and risk
- `calculators/` — self-contained rights-impact, procedural-readiness, governance-readiness, public-risk, and public-value calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, impact assessment notes, procurement notes, governance review, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and educational. They are not legal advice, compliance certification, civil-rights advice, public administration advice, procurement advice, or operational guidance.

# Procurement Notes

Public-sector algorithmic procurement should address:

- model and system documentation;
- audit rights;
- data ownership and export rights;
- change logs and update approval;
- performance and equity reporting;
- appeal and reason-code support;
- security and privacy obligations;
- interoperability;
- exit plans;
- vendor lock-in risk;
- public records and oversight requirements.

Agencies can outsource software, but not public responsibility.

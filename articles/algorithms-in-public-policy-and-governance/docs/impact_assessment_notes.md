# Impact Assessment Notes

A public algorithmic impact assessment should record:

- public purpose;
- legal authority;
- decision role;
- affected people and communities;
- data sources and provenance;
- variables and proxy review;
- rights impact;
- equity impact;
- public value;
- alternatives to automation;
- human review;
- notice, correction, appeal, and remedy;
- vendor responsibilities;
- monitoring and stop rules.

Impact assessment is meaningful only when it can modify, limit, pause, or reject deployment.

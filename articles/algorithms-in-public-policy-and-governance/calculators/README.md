# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `procedural_readiness_calculator.py` — average due process, transparency, human review, and appeal readiness.
- `governance_readiness_calculator.py` — average data quality, vendor accountability, monitoring, and procedural readiness.
- `public_algorithmic_risk_calculator.py` — rights impact multiplied by weak governance readiness.
- `eligibility_rule_calculator.py` — simple eligibility rule check.
- `public_value_calculator.py` — effectiveness, equity, accountability, and harm public-value score.

## R calculators

- `public_governance_score.R` — compute procedural readiness, governance readiness, and risk indicators.

Run smoke tests:

```bash
bash calculators/run_calculator_smoke_tests.sh
```

# Use Boundaries

This companion folder is educational. It should not be used as a production oversight certification system without domain validation.

Do not use the synthetic reliance or risk scores here for real procurement, compliance certification, safety approval, hiring, health, credit, public administration, or automated decision-making.

Before using oversight metrics in practice, require:

- domain-specific reliability evidence;
- user research;
- workflow analysis;
- legal and ethical review;
- appeal and correction testing;
- accountability documentation;
- stakeholder review.

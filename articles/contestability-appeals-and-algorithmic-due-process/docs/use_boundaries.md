# Use Boundaries

This companion folder is educational. It is not legal advice and should not be used as a compliance determination.

Do not use the synthetic contestability or procedural-risk scores here for real legal certification, public-benefits administration, credit, hiring, health, housing, policing, immigration, education, platform enforcement, or automated decision-making.

Before using contestability metrics in practice, require:

- legal review;
- domain-specific stakes assessment;
- stakeholder and affected-person review;
- accessibility testing;
- language and disability-access review;
- appeal outcome monitoring;
- correction and remediation procedures;
- institutional accountability.

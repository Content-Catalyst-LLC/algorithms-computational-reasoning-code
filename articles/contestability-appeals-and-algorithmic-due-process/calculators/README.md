# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `contestability_score_calculator.py` — average notice, reasons, evidence access, review, correction, and remedy.
- `procedural_risk_calculator.py` — combine stakes and weak contestability.
- `appeal_burden_calculator.py` — combine time, friction, language difficulty, and accessibility support.
- `due_process_review_trigger.py` — flag high-stakes, low-contestability review triggers.

## R calculators

- `contestability_summary_score.R` — compute contestability and procedural-risk scores.

Run smoke tests:

```bash
bash calculators/run_calculator_smoke_tests.sh
```

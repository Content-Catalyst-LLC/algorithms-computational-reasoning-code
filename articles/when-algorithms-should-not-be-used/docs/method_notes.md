# Method Notes

This folder treats non-use as a positive governance decision.

The core method is:

1. Define the proposed target, decision role, affected people, and stakes.
2. Assess target legitimacy, data legitimacy, proxy validity, and measurement limits.
3. Evaluate stakes, irreversibility, dignity, rights, contestability, and repairability.
4. Assess human judgment needs, governance capacity, monitoring, appeal, and stop authority.
5. Compare algorithmic use to improved human-led, participatory, policy, service, or non-automated alternatives.
6. Classify the decision: allow with controls, support-only, human-led, delay, refuse, pause, rollback, or retire.
7. Document reasons, evidence, owners, alternatives, and conditions for reconsideration.

The examples are dependency-light and educational.

# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `non_use_pressure_calculator.py` — average stakes, irreversibility, governance weakness, and proxy illegitimacy.
- `responsible_use_readiness_calculator.py` — average target legitimacy, data legitimacy, contestability, human judgment, governance capacity, and repairability.
- `repairability_calculator.py` — average detection, correction, remedy, and recurrence prevention.
- `refusal_threshold_calculator.py` — classify use based on non-use pressure and responsible-use readiness.

## R calculators

- `non_use_score.R` — compute non-use pressure, responsible-use readiness, and refusal status.

Run smoke tests:

```bash
bash calculators/run_calculator_smoke_tests.sh
```

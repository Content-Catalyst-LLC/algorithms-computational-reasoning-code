# AI Code Generation Caution Notes

AI code generation extends the desire for human-facing programming, but it is not the same as compilation.

Compilers:

- translate formal languages according to defined rules;
- can be tested against specifications;
- should preserve semantics under language rules.

AI coding tools:

- generate statistically likely code from prompts and context;
- can hallucinate APIs, logic, licenses, or security assumptions;
- require human review, tests, security analysis, and governance.

Hopper's legacy supports better tools, not unreviewed automation.

# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching compilers and automatic programming;
- explaining Hopper's role in human-readable programming;
- discussing A-0, FLOW-MATIC, COBOL, standards, validation, documentation, and debugging;
- comparing compiler automation with AI code-generation tools;
- building simple calculators and smoke tests for article concepts.

Do not treat synthetic scores as compiler conformance tests, formal semantic proofs, code quality certifications, or AI coding safety guarantees.

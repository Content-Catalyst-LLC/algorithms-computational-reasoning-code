# Documentation and Debugging Notes

Hopper's legacy includes software culture.

Key practices:

- manuals;
- examples;
- diagnostics;
- debugging logs;
- validation;
- standards documents;
- teaching and explanation.

The famous Mark II moth is not the origin of the word "bug," but it became a memorable artifact of debugging culture.

Software becomes human when it can be explained, repaired, taught, and inherited.

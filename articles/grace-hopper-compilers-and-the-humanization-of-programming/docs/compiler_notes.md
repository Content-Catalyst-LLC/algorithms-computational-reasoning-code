# Compiler Notes

A compiler translates source language into target code or another executable/intermediate representation.

Teaching pipeline:

`Source -> Tokens -> Syntax Tree -> Checked Program -> Target Code`

Core responsibilities:

- recognize valid syntax;
- enforce language rules;
- translate reliably;
- report understandable errors;
- support portability where possible;
- preserve semantics under translation;
- enable testing and validation.

Compilers are human-machine bridges.

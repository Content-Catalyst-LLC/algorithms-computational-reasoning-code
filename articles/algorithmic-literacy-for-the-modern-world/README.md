# Algorithmic Literacy for the Modern World

Companion code and reproducible workflows for **Algorithmic Literacy for the Modern World**.

This article examines algorithmic literacy as a public, educational, technical, civic, and institutional capacity for recognizing, interpreting, questioning, and responsibly using algorithmic systems.

## Folder structure

```text
python/      literacy audits, CLI tools, calculators, tests
r/           statistical summaries, visualization, reports
julia/       literacy-gap simulations and procedural-visibility examples
sql/         schemas, audit tables, literacy queries
haskell/     typed literacy dimensions and procedural contracts
rust/        systems-style literacy diagnostics
go/          CLI/service-style literacy workflows
c/           low-level diagnostic examples
cpp/         high-performance diagnostic examples
fortran/     literacy-gap numerical examples
java/        object-oriented literacy audit systems
typescript/ interface-facing literacy and transparency checks
prolog/      rule-based literacy logic
racket/      interpreter-style literacy examples
docs/        methodology, governance, and article notes
data/        synthetic datasets only
outputs/     generated outputs
notebooks/   notebook-ready walkthroughs
canvas/      Catalyst Canvas-ready cards and manifests
shared/      schemas, templates, taxonomies, benchmarks, governance
```

## Run

```bash
make smoke
make all
```

The workflows use synthetic data only and are intended for educational, reproducible article support.

# Algorism and Place Value Notes

Algorism refers historically to written arithmetic with Hindu-Arabic numerals.

Important procedural features:

- digits compactly represent numbers;
- place value makes position meaningful;
- zero marks absence in a position;
- carrying moves value across places;
- borrowing transforms value across places;
- written layout provides external memory;
- checking supports verification.

Algorism is algorithmic because representation and method work together.

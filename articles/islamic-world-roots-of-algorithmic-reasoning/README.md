# Islamic-World Roots of Algorithmic Reasoning

This companion folder supports the Sustainable Catalyst article **Islamic-World Roots of Algorithmic Reasoning**.

It provides reproducible, teaching-oriented workflows for mapping Islamic-world roots of algorithmic reasoning: al-Khwārizmī, algorism, algebra, Hindu-Arabic numerals, positional calculation, verbal procedures, practical mathematics, inheritance calculation, commerce, surveying, astronomical tables, calendars, geography, coordinates, cryptanalysis, mechanical devices, translation movements, Latin reception, origin-story cautions, and historical computational reasoning.

## What this folder demonstrates

- how historical algorithmic-reasoning themes can be mapped across procedural explicitness, transmission importance, practical application, representational importance, and modern computational resonance;
- how careful historical interpretation can distinguish etymology from total origin, transmission from passive preservation, and pre-computer procedure from modern programming;
- how positional calculation, algebraic transformation, tabular computation, cryptanalytic frequency analysis, practical calculation, and mechanical sequencing can be represented as computational patterns;
- how calculators can demonstrate place value, algebraic balancing, table interpolation, frequency-distance comparison, inheritance-share normalization, and historical significance scoring;
- how documentation can support accurate, appreciative, non-reductive historical interpretation.

## Repository structure

- `python/` — reference historical algorithmic-reasoning map workflow and tests
- `r/` — historical theme diagnostics and figures
- `sql/` — schema and queries for historical themes, significance scores, and cautions
- `calculators/` — self-contained place-value, algebra-balancing, interpolation, frequency-distance, inheritance-share, and historical-significance calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, origin-story cautions, transmission notes, terminology notes, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and interpretive. They are educational scaffolds, not claims of exhaustive historical proof.

# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `place_value_calculator.py` — compute digit × base^position.
- `algebra_balance_calculator.py` — apply the same addition to both sides of an equation.
- `table_interpolation_calculator.py` — linear interpolation between two table values.
- `frequency_distance_calculator.py` — compare observed and expected frequencies.
- `inheritance_share_normalizer.py` — normalize raw shares so they sum to one.
- `historical_significance_calculator.py` — average interpretive theme dimensions.

## R calculators

- `historical_significance_score.R` — compute an interpretive significance score.

Run smoke tests:

```bash
bash calculators/run_calculator_smoke_tests.sh
```

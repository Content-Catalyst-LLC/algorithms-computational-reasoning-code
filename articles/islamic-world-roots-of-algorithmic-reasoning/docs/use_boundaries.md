# Use Boundaries

This companion folder is educational and interpretive. It is not a complete historical database, not a philological edition, not a manuscript catalogue, not a substitute for specialist scholarship, and not proof of any single-civilization origin claim.

Use these files to support:

- teaching;
- conceptual mapping;
- historical framing;
- algorithmic-reasoning interpretation;
- structured comparison of themes;
- computational scaffolds for humanities-oriented analysis.

Do not use them to make unsupported claims that algorithms began in only one civilization or that medieval procedures were equivalent to modern computer programs.

# Simulation Calculators

Calculator scripts for the article **Simulation as Computational Reasoning**.

They demonstrate:

- state updates over time;
- intervention strength and final stock;
- simple deterministic trajectories;
- reproducible output tables for future web-tool adaptation.

Run:

```bash
python3 calculators/python/simulation_calculator.py
Rscript calculators/r/simulation_calculator.R
```

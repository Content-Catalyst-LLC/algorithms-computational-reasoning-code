# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not production benchmarks, database design tools, or AI infrastructure audits.

## Python calculators

- `history_score_calculator.py` — average data-structure/analysis history dimensions.
- `growth_rate_calculator.py` — compare constant, log, linear, n log n, and quadratic costs.
- `binary_search_steps_calculator.py` — estimate binary-search halving steps.
- `graph_memory_calculator.py` — compare adjacency matrix and adjacency list storage.
- `hash_table_load_calculator.py` — estimate load factor and rough risk status.
- `amortized_array_append_calculator.py` — estimate amortized cost for dynamic-array appends.
- `priority_queue_operation_calculator.py` — estimate heap operation cost.
- `ai_structure_audit_calculator.py` — flag AI infrastructure claims that ignore data-structure governance.

## R calculators

- `history_score.R` — compute data-structure/analysis history score in R.

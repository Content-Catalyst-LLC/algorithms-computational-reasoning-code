# AI Infrastructure Notes

AI systems depend on data structures.

Common AI infrastructure structures:

- tensors;
- sparse matrices;
- token sequences;
- document indexes;
- vector embeddings;
- approximate nearest-neighbor indexes;
- knowledge graphs;
- queues;
- caches;
- logs;
- traces;
- provenance metadata;
- evaluation tables;
- confusion matrices;
- model-card records.

AI governance requires data-structure audits because representation choices influence retrieval quality, bias, performance, reproducibility, monitoring, and accountability.

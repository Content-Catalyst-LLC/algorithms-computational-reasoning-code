# Methodology

This article folder treats linear programming and convex optimization as both mathematical and governance workflows.

The examples use synthetic data and transparent calculations to show:

1. how decision variables define the available choices;
2. how objectives define what counts as better;
3. how constraints define feasibility;
4. how slack and sensitivity reveal scarce resources;
5. how convexity supports global-optimality reasoning;
6. how governance review asks whether the modeled optimum is legitimate in context.

The Python and R workflows intentionally avoid heavy dependencies so the examples can run in constrained environments.

# Responsible Use Notes

Optimization models should not be treated as neutral merely because they are mathematical. Before using an optimal result in an institutional setting, review:

- whether the objective reflects a legitimate purpose;
- whether constraints include legal, safety, ethical, fairness, and operational limits;
- whether coefficients are traceable to reliable data;
- whether uncertainty and sensitivity have been examined;
- whether burdens and benefits are distributed responsibly;
- whether human review and exception paths exist;
- whether the model communicates assumptions and limitations clearly.

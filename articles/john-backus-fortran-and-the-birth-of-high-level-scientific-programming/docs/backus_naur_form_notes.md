# Backus-Naur Form Notes

Backus-Naur Form provides a formal notation for describing programming-language syntax.

Key ideas:

- nonterminals represent syntactic categories;
- terminals represent source tokens;
- production rules define allowed forms;
- recursive grammar defines nested language structure;
- formal syntax supports compiler design, standards, and teaching.

Backus contributed both to practical high-level programming and to formal language specification.

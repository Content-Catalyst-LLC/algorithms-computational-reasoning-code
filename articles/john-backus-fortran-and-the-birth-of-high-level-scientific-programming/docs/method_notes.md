# Method Notes

This folder treats Backus and FORTRAN as the practical birth of high-level scientific programming.

The core method is:

1. Begin with machine-specific scientific programming before Fortran.
2. Study formula translation as the core language idea.
3. Place FORTRAN in the IBM 704 hardware and user context.
4. Analyze compiler optimization as the key to adoption.
5. Study arrays, loops, and subroutines as scientific structures.
6. Trace portability, standards, and scientific software continuity.
7. Connect Backus-Naur Form to language specification.
8. Apply the Fortran lesson to AI-generated scientific code.

The goal is to understand abstraction, performance, and scientific expression as a linked historical achievement.

# Fortran and High-Performance Computing Notes

Fortran persists in scientific and high-performance computing because of:

- numerical performance;
- mature optimizing compilers;
- array-oriented scientific work;
- long-lived models and libraries;
- institutional codebases;
- portability through standards;
- scientific trust accumulated over decades.

Fortran is not only a legacy language. It is a case study in how a language can become scientific infrastructure.

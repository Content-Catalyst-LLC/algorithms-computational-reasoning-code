# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching formula translation;
- explaining FORTRAN's role in scientific computing;
- reviewing compiler optimization and performance credibility;
- discussing arrays, loops, subroutines, and numerical work;
- connecting Backus to BNF and later functional-programming ideas;
- reviewing generated scientific code.

Do not treat synthetic scores as compiler conformance tests, numerical certifications, production benchmarks, or safety guarantees.

# Formula Translation Notes

FORTRAN means Formula Translation.

Central idea:

`scientific formula -> high-level source -> compiler -> machine code`

Formula translation requires:

- arithmetic expressions;
- operator precedence;
- variables;
- arrays;
- loops;
- subroutines;
- parsing;
- code generation;
- optimization.

The historical importance is that scientific users could express procedures in a form closer to their domain while relying on the compiler to generate efficient machine instructions.

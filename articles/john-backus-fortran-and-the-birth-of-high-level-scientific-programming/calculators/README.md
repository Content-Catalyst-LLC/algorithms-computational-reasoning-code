# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not compiler validation or numerical safety tools.

## Python calculators

- `birth_score_calculator.py` — average Backus/FORTRAN dimensions.
- `formula_translation_calculator.py` — evaluate y = ax^2 + bx + c.
- `vector_loop_calculator.py` — apply sin(x) + x^2 over comma-separated values.
- `portability_score_calculator.py` — estimate source portability across target compilers.
- `performance_credibility_calculator.py` — compare high-level generated runtime with hand-coded baseline.
- `translation_trust_calculator.py` — combine readability, correct translation, performance, tests, and documentation.
- `bnf_rule_helper.py` — produce a simple BNF-style production.
- `ai_scientific_code_caution_calculator.py` — flag generated-scientific-code overclaims.

## R calculators

- `birth_score.R` — compute high-level scientific programming birth score.

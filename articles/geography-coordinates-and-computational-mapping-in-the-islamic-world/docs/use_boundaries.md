# Use Boundaries

This companion folder is educational and interpretive. It is not navigational, surveying, religious, cartographic, legal, geographic, administrative, or travel advice.

Use these files to support:

- teaching;
- conceptual mapping;
- historical framing;
- coordinate and route analysis;
- structured comparison of mapping themes;
- computational scaffolds for humanities-oriented analysis.

Do not use the calculators for real navigation, qibla determination, surveying, GIS production, or administrative decisions.

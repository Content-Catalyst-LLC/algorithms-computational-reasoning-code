# Coordinate Data Notes

Coordinates make place calculable.

Important interpretive points:

- Latitude and longitude create a common frame for comparison.
- Coordinate lists can function as structured spatial datasets.
- Reference systems matter.
- Measurement and copying introduce uncertainty.
- Coordinates can support plotting, comparison, correction, and reconstruction.
- Coordinates are structured but not context-free.

A coordinate list is a premodern spatial data artifact, not a neutral mirror of the world.

# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `coordinate_record_calculator.py` — create a structured place-coordinate record.
- `approx_coordinate_distance_calculator.py` — approximate distance between nearby coordinates.
- `route_distance_calculator.py` — sum route segment distances.
- `bearing_calculator.py` — compute an approximate bearing between two coordinate points.
- `climate_band_classifier.py` — classify a synthetic climate band from latitude.
- `projection_scale_calculator.py` — convert real-world distance to map distance by scale.
- `mapping_score_calculator.py` — average spatial representation, coordinate structure, procedure, institution, correction, transmission, and modern resonance.

## R calculators

- `mapping_score.R` — compute computational mapping score.

# Geography, Coordinates, and Computational Mapping in the Islamic World

This companion folder supports the Sustainable Catalyst article **Geography, Coordinates, and Computational Mapping in the Islamic World**.

It provides reproducible, teaching-oriented workflows for mapping geography, coordinates, route networks, qibla-style directional reasoning, al-Khwārizmī’s geographical tables, al-Idrīsī’s cartographic synthesis, climate zones, projection and scale choices, spatial data correction, manuscript transmission, and computational mapping as spatial algorithmic reasoning.

## What this folder demonstrates

- how coordinates turn place into structured spatial data;
- how geographic tables, gazetteers, and coordinate lists function as premodern spatial databases;
- how routes can be modeled as networks of places, distances, stages, and constraints;
- how maps require projection, scale, orientation, classification, omission, and correction;
- how Islamic-world geographic traditions inherited, corrected, extended, and synthesized earlier materials;
- how qibla, timekeeping, travel, administration, commerce, and scholarship connect geography to procedure;
- how spatial computation predates digital GIS while remaining historically distinct from it.

## Repository structure

- `python/` — reference computational mapping procedure map workflow and tests
- `r/` — computational mapping diagnostics and figures
- `sql/` — schema and queries for mapping themes, scores, and interpretation cautions
- `calculators/` — self-contained coordinate, distance, route, bearing, region, projection-scale, and mapping-score calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, coordinate notes, route-network notes, qibla/direction notes, map correction notes, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and interpretive. They are educational scaffolds, not navigational, surveying, legal, geographic, religious, or administrative advice.

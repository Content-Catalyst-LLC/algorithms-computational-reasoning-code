# Method Notes: Bayesian Computation and Updating Beliefs

This folder uses a beta-binomial teaching example because it makes the core Bayesian workflow easy to audit: prior, evidence, posterior, credible interval, threshold probability, prior sensitivity, and expected-loss decision summary.

Review questions: What belief target is being updated? What prior was used and why? What evidence was observed? What likelihood connects evidence to belief? How uncertain is the posterior? Does the result depend on prior choice? What decision rule uses the posterior? What validation and governance evidence is missing?

# Calculators

These small calculators support article concepts without requiring external packages.

## Python calculators

- `symbolic_rule_calculator.py` — apply simple if-all-premises-present rule.
- `constraint_satisfaction_calculator.py` — check whether required constraints are present.

## R calculators

- `rule_trace_score.R` — summarize trace completeness from a small CSV-style input.

Run smoke tests:

```bash
bash calculators/run_calculator_smoke_tests.sh
```

# Automated Reasoning, Symbolic AI, and Hybrid Systems

This companion folder supports the Sustainable Catalyst article **Automated Reasoning, Symbolic AI, and Hybrid Systems**.

It provides reproducible, teaching-oriented workflows for automated reasoning, symbolic AI, inference traces, rule provenance, satisfiability-style constraint checks, contradiction review, logic programming, hybrid/neuro-symbolic interface review, governance artifacts, and responsible computational interpretation.

## What this folder demonstrates

- how facts and rules produce derived conclusions;
- how symbolic inference traces can support review;
- how contradiction checks and constraint satisfaction summaries can be documented;
- how hybrid systems should track the interface between learned outputs and symbolic rules;
- how automated reasoning workflows can be governed through provenance, use boundaries, and contestability;
- how lightweight calculators can support rule and constraint review.

## Repository structure

- `python/` — reference symbolic-reasoning audit workflow and tests
- `r/` — diagnostic summaries and figures
- `sql/` — schema and governance queries for facts, rules, traces, and constraints
- `calculators/` — self-contained rule, entailment, and constraint calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, governance review, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and educational. They are not intended for operational deployment.

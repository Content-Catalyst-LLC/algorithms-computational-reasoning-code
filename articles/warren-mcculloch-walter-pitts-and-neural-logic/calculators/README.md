# Calculators

These calculators support the neural logic article without external packages.

## Python calculators

- `threshold_unit_calculator.py` — compute a binary threshold-unit output.
- `and_gate_threshold_calculator.py` — demonstrate AND as threshold logic.
- `or_gate_threshold_calculator.py` — demonstrate OR as threshold logic.
- `neural_logic_score_calculator.py` — average neural-logic concept dimensions.
- `biological_caution_calculator.py` — estimate abstraction caution from biological simplification.
- `ai_lineage_boundary_calculator.py` — flag overclaims about modern deep learning.
- `network_state_update_calculator.py` — update a tiny threshold network.
- `responsible_abstraction_calculator.py` — score formal clarity, stated limits, and historical care.

## R calculators

- `neural_logic_score.R` — compute a neural-logic score in R.

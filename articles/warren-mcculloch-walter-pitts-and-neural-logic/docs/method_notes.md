# Method Notes

This folder treats McCulloch-Pitts neural logic as a historical bridge between nervous activity, symbolic logic, cybernetics, automata, and artificial intelligence.

The core method is:

1. Identify the biological inspiration.
2. Define the formal abstraction.
3. Map threshold rules and logical operations.
4. Trace network behavior over time.
5. Connect neural logic to cybernetics and automata.
6. Distinguish the model from modern deep learning.
7. State biological and philosophical limits.
8. Connect abstraction boundaries to responsible AI.

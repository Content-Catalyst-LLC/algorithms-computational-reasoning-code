# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching McCulloch-Pitts neural logic;
- explaining threshold units and Boolean networks;
- connecting cybernetics, automata, and AI history;
- discussing symbolic/neural boundaries;
- reviewing abstraction limits in AI.

Do not treat this material as neuroscience validation, model certification, production AI guidance, medical advice, or safety approval.

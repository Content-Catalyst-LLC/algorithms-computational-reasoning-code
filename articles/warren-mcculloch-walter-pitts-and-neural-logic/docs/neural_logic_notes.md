# Neural Logic Notes

Neural logic is the idea that simplified neural activity can be represented through formal logical relations.

Key ideas:

- simplified neurons as threshold units;
- binary firing as an abstraction;
- networks of units as formal circuits;
- inhibition as negation-like control;
- activity over time as procedure;
- state transitions as automata-like behavior;
- cybernetic relations among signal, control, and feedback.

The McCulloch-Pitts model is historically powerful because it made neural networks formal. It is also limited because real neurons, brains, and cognition exceed the abstraction.

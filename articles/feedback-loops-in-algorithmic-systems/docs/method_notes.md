# Method Notes

This folder treats feedback loops as a core part of algorithmic-system behavior.

The core method is:

1. Map the feedback path from output to future input.
2. Track exposure, ranking, recommendation, or intervention.
3. Identify amplification and concentration effects.
4. Monitor drift, recursive data, and behavioral adaptation.
5. Separate natural observations from model-mediated observations.
6. Establish correction pathways and update boundaries.
7. Review feedback loops over time.

The examples are dependency-light and educational.

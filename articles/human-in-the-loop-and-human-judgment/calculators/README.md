# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `review_capacity_calculator.py` — average time, information, authority, training, and protection.
- `reliance_calculator.py` — accepted recommendations divided by total recommendations.
- `override_rate_calculator.py` — overrides divided by reviewed cases.
- `review_risk_calculator.py` — stakes multiplied by weak judgment capacity.

## R calculators

- `human_review_score.R` — compute review capacity, reliance, override rate, judgment capacity, and review risk.

Run smoke tests:

```bash
bash calculators/run_calculator_smoke_tests.sh
```

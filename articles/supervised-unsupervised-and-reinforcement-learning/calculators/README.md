# Calculators

Self-contained calculators for the **Supervised, Unsupervised, and Reinforcement Learning** companion folder.

Examples:

```bash
python3 calculators/python/classification_metrics_calculator.py --tp 42 --fp 8 --tn 37 --fn 13
python3 calculators/python/generalization_gap_calculator.py --train-metric 0.82 --test-metric 0.74
python3 calculators/python/cluster_assignment_calculator.py --x 0.44 --y 0.61
python3 calculators/python/expected_reward_calculator.py --reward-probability 0.54 --reward-value 1 --cost 0.08
```

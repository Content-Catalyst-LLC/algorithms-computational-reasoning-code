# Terminology and Vocabulary Notes

Technical translation depends on vocabulary choices.

A translator may:

- borrow a source term;
- create a calque;
- coin a new technical term;
- add a gloss;
- stabilize a term through repeated usage;
- distinguish near synonyms;
- adapt a concept to a new disciplinary environment.

For computational knowledge, vocabulary is infrastructure. If technical terms fail, procedures become unstable.

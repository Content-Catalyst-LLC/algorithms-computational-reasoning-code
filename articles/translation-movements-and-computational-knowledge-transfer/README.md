# Translation Movements and Computational Knowledge Transfer

This companion folder supports the Sustainable Catalyst article **Translation Movements and Computational Knowledge Transfer**.

It provides reproducible, teaching-oriented workflows for mapping translation movements as computational knowledge infrastructure: procedural fidelity, technical vocabulary, diagrams, tables, commentary, correction, adaptation, multilingual relay, institutional support, and error control.

## What this folder demonstrates

- how technical translation preserves procedures, not only prose;
- how mathematical, astronomical, mechanical, medical, and philosophical knowledge travels through texts, tables, diagrams, examples, instruments, and teaching contexts;
- how Greek, Syriac, Persian, Sanskrit, Arabic, Hebrew, and Latin channels function as multilingual relay systems;
- how terminology, notation, units, diagrams, and commentary shape computational knowledge transfer;
- how translation can preserve, correct, adapt, and transform procedural knowledge;
- how single-origin stories oversimplify the intercultural history of algorithmic reasoning.

## Repository structure

- `python/` — reference computational knowledge transfer map workflow and tests
- `r/` — transfer diagnostics and figures
- `sql/` — schema and queries for transfer themes, scores, and interpretation cautions
- `calculators/` — self-contained procedural-fidelity, vocabulary-map, table-error, relay-depth, transfer-score, and knowledge-object calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, terminology notes, diagram/table notes, institutions notes, commentary/error-control notes, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and interpretive. They are educational scaffolds for historical and conceptual analysis, not claims about any single manuscript lineage.

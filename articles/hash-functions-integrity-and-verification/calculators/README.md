# Calculators: Hash Functions, Integrity, and Verification

This calculator layer provides self-contained examples for digest comparison, tamper detection, and hash-verification governance.

- `python/hash_verification_calculator.py` uses Python standard-library cryptographic hashes.
- `r/hash_verification_calculator.R` uses a small teaching checksum only to demonstrate comparison logic in base R.
- `run_calculator_smoke_tests.sh` runs the available calculator examples.

# Data Dictionary

- `model_error_synthetic_observations.csv` — synthetic train, test, and shifted-test observations.
- `model_complexity_metrics.csv` — error metrics by model capacity and regularization level.
- `model_error_validation_curve.csv` — train/test error by polynomial degree.
- `model_error_learning_curve.csv` — train/test error by training-set size.
- `model_error_residual_diagnostics.csv` — residual summaries by split and synthetic region.
- `model_error_governance_review.csv` — review questions for model-error interpretation.

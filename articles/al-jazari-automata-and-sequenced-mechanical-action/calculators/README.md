# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not engineering tools.

## Python calculators

- `event_trigger_calculator.py` — determine whether a level, time, or cam position reaches a trigger.
- `cycle_time_calculator.py` — estimate cycle time from stages.
- `cam_schedule_calculator.py` — report whether a rotating cam has reached an activation angle.
- `repeat_cycle_calculator.py` — compute repeated-cycle output count.
- `water_flow_time_calculator.py` — estimate time from volume and flow rate.
- `reset_state_calculator.py` — reset a toy state to an initial condition.
- `sequenced_action_score_calculator.py` — average sequence, timing, embodiment, conditional action, repeatability, documentation, history, caution, and modern resonance.

## R calculators

- `sequenced_action_score.R` — compute sequenced action score.

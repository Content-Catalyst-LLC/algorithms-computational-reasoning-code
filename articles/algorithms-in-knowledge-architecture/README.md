# Algorithms in Knowledge Architecture

This companion folder supports the Sustainable Catalyst article **Algorithms in Knowledge Architecture**.

It provides reproducible, teaching-oriented workflows for classification, metadata, taxonomy, ontology, search, ranking, retrieval, recommendation, internal linking, semantic networks, knowledge graphs, provenance, freshness, editorial review, maintenance, and knowledge-system governance.

## Article sequence

- Previous: `algorithms-in-systems-modeling`
- Current: `algorithms-in-knowledge-architecture`
- Next: `algorithms-in-public-policy-and-governance`

## What this folder demonstrates

- how metadata quality, taxonomy fit, search readiness, link quality, recommendation quality, provenance, freshness, editorial review, and representation risk can be audited;
- how architecture readiness and maintenance risk can support editorial governance;
- how search, linking, recommendation, and metadata workflows should be documented and reviewed;
- how knowledge architecture can support discovery without collapsing knowledge value into engagement signals;
- how calculators, tests, SQL queries, Canvas cards, and multilingual examples can support scalable knowledge systems.

## Repository structure

- `python/` — reference knowledge-architecture audit workflow and tests
- `r/` — diagnostic summaries and figures
- `sql/` — schema and queries for metadata, search, linking, recommendation, maintenance, and governance
- `calculators/` — self-contained metadata, readiness, maintenance-risk, relevance, similarity, and link-score calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, taxonomy notes, metadata schema notes, governance review, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and educational. They are not compliance, records-management, library-science, legal, archival, publishing, or operational advice.

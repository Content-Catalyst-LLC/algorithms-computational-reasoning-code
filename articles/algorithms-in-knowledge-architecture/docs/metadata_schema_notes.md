# Metadata Schema Notes

A metadata schema should define:

- title;
- slug or identifier;
- author or source;
- publication or creation date;
- modified date;
- topic categories;
- tags;
- series or collection;
- difficulty or audience;
- related items;
- source provenance;
- review status;
- version;
- license;
- repository link when applicable.

Metadata should make content findable, maintainable, auditable, and reusable.

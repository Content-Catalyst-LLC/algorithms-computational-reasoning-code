# Use Boundaries

This companion folder is educational. It is not legal advice, compliance certification, records-management certification, library-science certification, archival advice, publishing advice, SEO advice, or operational guidance.

Do not use the synthetic knowledge-architecture scores here to certify real search systems, recommendation systems, metadata systems, or institutional repositories.

Before using knowledge-architecture metrics in practice, require:

- domain expertise;
- editorial review;
- metadata governance;
- search-quality evaluation;
- provenance review;
- representation-risk review;
- user research where appropriate;
- accessibility review;
- lifecycle maintenance and deprecation rules.

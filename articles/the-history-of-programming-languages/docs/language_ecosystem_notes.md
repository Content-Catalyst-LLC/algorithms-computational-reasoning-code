# Language Ecosystem Notes

A programming language is more than syntax.

A language ecosystem includes:

- compiler or interpreter;
- runtime;
- standard library;
- package manager;
- third-party packages;
- build tools;
- testing frameworks;
- documentation;
- editors and language servers;
- style guides;
- standards bodies;
- communities;
- institutions that depend on the language.

A technically elegant language can remain niche if its ecosystem does not fit user needs. A messy language can become dominant when it solves practical problems and develops strong infrastructure.

# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not language rankings, compiler tests, or security audits.

## Python calculators

- `history_score_calculator.py` — average programming-language history dimensions.
- `paradigm_classifier.py` — rough classifier for language-tradition descriptions.
- `ecosystem_score_calculator.py` — estimate ecosystem strength from tools and community factors.
- `type_safety_score_calculator.py` — estimate type/safety orientation.
- `translation_risk_calculator.py` — estimate risk across source, translator, runtime, and dependencies.
- `dependency_governance_calculator.py` — estimate package/dependency governance risk.
- `ai_language_layer_caution_calculator.py` — flag AI-code claims that ignore language ecosystems.
- `language_comparison_helper.py` — compare two traditions across selected dimensions.

## R calculators

- `history_score.R` — compute language-history score in R.

# Governance Review

Algorithmic accountability governance should include:

- named system owner;
- named data steward;
- named model owner;
- approval gates before deployment;
- audit trail requirements;
- monitoring thresholds and escalation rules;
- appeal and correction pathways;
- incident response and evidence preservation;
- remediation and recurrence prevention;
- authority to pause, rollback, modify, or retire the system.

Governance should define not only who can approve automation, but who can stop it.

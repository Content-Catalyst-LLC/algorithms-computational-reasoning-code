# Governance and Use Boundary

This companion repository treats causal inference as accountable computational reasoning.

The workflow should not be used to justify real institutional action without domain review. Its outputs are examples of how to document causal assumptions, not proof that a real intervention works.

## Acceptable use

- Teaching causal inference concepts.
- Demonstrating the distinction between association and intervention.
- Showing how confounding can bias naive estimates.
- Prototyping audit tables and documentation templates.
- Building future website calculators from transparent examples.

## Not acceptable without additional review

- Operational policy decisions.
- Clinical, legal, financial, educational, or employment decisions.
- Automated eligibility or risk scoring.
- Claims about real-world treatment effects.
- Claims that prediction accuracy implies causation.

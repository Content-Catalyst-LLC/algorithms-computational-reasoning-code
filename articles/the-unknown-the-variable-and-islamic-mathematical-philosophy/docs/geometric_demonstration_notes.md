# Geometric Demonstration Notes

Geometric demonstration helps justify algebraic procedure.

Key ideas:

- a root can be interpreted as a line segment;
- a square can be interpreted as an area;
- completing a square can be made visible by construction;
- equality can be shown through equivalent magnitudes;
- proof gives a procedure more than instrumental usefulness;
- verification checks that the derived value satisfies the original condition.

Geometry helped make algebraic transformation credible and demonstrable.

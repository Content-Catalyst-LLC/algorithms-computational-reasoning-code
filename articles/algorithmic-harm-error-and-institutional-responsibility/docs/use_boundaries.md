# Use Boundaries

This companion folder is educational. It is not legal, compliance, procurement, safety, or operational advice.

Do not use the synthetic harm-risk or responsibility scores here to certify real systems, approve deployments, decide procurement, satisfy regulatory requirements, or automate decisions.

Before using harm and responsibility metrics in practice, require:

- legal review;
- domain expertise;
- affected-person and stakeholder consultation;
- validated data;
- incident response procedures;
- remediation authority;
- accessibility and appeal testing;
- independent audit where appropriate.

# Notebook Layer

This folder is reserved for notebook-ready versions of the article workflows.

Suggested notebook sequence:

1. Generate synthetic intervention cases.
2. Compare modeled intervention scenarios.
3. Review adjustment sets and causal assumptions.
4. Test threshold sensitivity.
5. Compare effect, cost, net benefit, and governance risk.
6. Write a short interpretation memo with use boundaries.

The repository keeps the executable source in `python/` so notebooks can remain thin, readable teaching layers.

# From Baghdad to Latin Europe: Algorism, Algebra, and Reception

This companion folder supports the Sustainable Catalyst article **From Baghdad to Latin Europe: Algorism, Algebra, and Reception**.

It provides reproducible, teaching-oriented workflows for mapping algorism, algebra, and Latin reception as computational knowledge transfer: Hindu-Arabic numerals, place value, zero, arithmetic procedures, Arabic-Latin translation, Toledo, Fibonacci, *Liber Abaci*, abacus culture, merchant arithmetic, institutional adoption, terminology, resistance, trust, and reception.

## What this folder demonstrates

- how algorism functioned as received written arithmetic procedure;
- how algebra moved as rule-governed problem solving before modern symbolism;
- how Hindu-Arabic numerals, zero, and place value reshaped representation;
- how Arabic-Latin translation, Toledo, schools, merchants, manuscripts, and universities shaped reception;
- how Fibonacci and *Liber Abaci* belong to a broader network of mathematical popularization and pedagogy;
- how abacus culture and older practices interacted with written algorism;
- how trust, verification, institutional legitimacy, and commercial use determine whether computational methods spread;
- how origin stories should avoid both passive-borrowing and isolated-invention narratives.

## Repository structure

- `python/` — reference algorism/algebra/reception map workflow and tests
- `r/` — reception diagnostics and figures
- `sql/` — schema and queries for reception themes, scores, and interpretation cautions
- `calculators/` — self-contained place-value, carrying, borrowing, reception-score, trust, adoption, and algebraic-case calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, algorism notes, algebra reception notes, Fibonacci notes, abacus/trust notes, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and interpretive. They are educational scaffolds for historical and conceptual analysis, not claims about any single manuscript lineage or universal chronology.

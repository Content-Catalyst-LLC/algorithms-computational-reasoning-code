# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not historical proof tools.

## Python calculators

- `place_value_calculator.py` — compute value from comma-separated digits.
- `carrying_calculator.py` — demonstrate carry from ones to tens.
- `borrowing_calculator.py` — demonstrate borrowing in two-digit subtraction.
- `algebra_case_calculator.py` — classify a simple quadratic-like verbal algebra case.
- `reception_score_calculator.py` — average reception-dimension scores.
- `trust_verification_calculator.py` — score method trust from checks, teaching, adoption, auditability, and use.
- `adoption_stage_calculator.py` — classify adoption from novelty to normalization.

## R calculators

- `reception_score.R` — compute algorism/algebra reception score.

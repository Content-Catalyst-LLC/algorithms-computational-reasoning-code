# Algorithm Analysis Notes

Algorithm analysis studies the cost and behavior of procedures.

Questions:

- What problem is being solved?
- What data representation is assumed?
- What operations are counted?
- What is the worst case?
- What is the average case?
- How does cost grow with input size?
- What machine or abstract model is being assumed?
- What hidden constants or practical details matter?

Knuthian analysis combines mathematics, examples, implementation, and exposition.

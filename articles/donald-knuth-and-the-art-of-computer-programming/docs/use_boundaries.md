# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching algorithm analysis;
- comparing growth rates;
- discussing sorting/searching and lower bounds;
- explaining literate programming;
- reviewing AI-generated code for explanation and maintainability;
- building small calculators and smoke tests.

Do not treat synthetic scores as formal algorithm proofs, production benchmarks, conformance tests, security reviews, or software safety guarantees.

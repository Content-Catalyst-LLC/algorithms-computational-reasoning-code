# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not formal proofs or production benchmarks.

## Python calculators

- `art_score_calculator.py` — average Knuth algorithmic-art dimensions.
- `growth_rate_calculator.py` — compare common asymptotic growth expressions.
- `comparison_sort_lower_bound.py` — compute log2(n!) lower-bound intuition.
- `runtime_comparison_helper.py` — compare two cost formulas over sample sizes.
- `literate_programming_split_helper.py` — sketch weave/tangle outputs.
- `tex_clarity_helper.py` — score technical-presentation clarity.
- `documentation_score_calculator.py` — estimate explanation and maintainability quality.
- `ai_code_review_caution_calculator.py` — flag generated-code overclaims.

## R calculators

- `art_score.R` — compute Knuth algorithmic-art score.

# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `place_value_calculator.py` — compute digit × base^position.
- `positional_expansion_calculator.py` — expand a base-10 integer into place-value terms.
- `carry_step_calculator.py` — compute digit and carry from a local sum.
- `borrow_step_calculator.py` — compute a simple borrow transformation.
- `base_conversion_calculator.py` — convert a nonnegative integer to another base.
- `zero_role_classifier.py` — classify zero as placeholder, arithmetic value, or boundary case.
- `positional_score_calculator.py` — average representation, procedure, transmission, practical use, pedagogy, and modern resonance.

## R calculators

- `positional_score.R` — compute positional calculation score.

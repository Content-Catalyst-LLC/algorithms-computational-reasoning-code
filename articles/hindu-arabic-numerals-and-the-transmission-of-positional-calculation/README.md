# Hindu-Arabic Numerals and the Transmission of Positional Calculation

This companion folder supports the Sustainable Catalyst article **Hindu-Arabic Numerals and the Transmission of Positional Calculation**.

It provides reproducible, teaching-oriented workflows for mapping Hindu-Arabic numerals, decimal place value, zero, carrying, borrowing, written reckoning, algorism, commercial calculation, astronomical tables, Latin reception, numeral-form variation, and positional calculation as algorithmic reasoning.

## What this folder demonstrates

- how positional notation can be studied as representational infrastructure for computation;
- how zero functions as placeholder, structural marker, arithmetic object, and boundary case;
- how carrying and borrowing are local rule-governed procedures over place-value states;
- how written algorism externalizes arithmetic for teaching, checking, copying, and transmission;
- how Indian mathematical origins, Arabic and Islamic-world scholarship, Latin reception, commercial use, and pedagogy together shaped the transmission of positional calculation;
- how careful interpretation avoids erasing either Indian origins or Arabic/Islamic-world adaptation and transmission.

## Repository structure

- `python/` — reference positional calculation transmission map workflow and tests
- `r/` — positional calculation diagnostics and figures
- `sql/` — schema and queries for positional themes, scores, and interpretation cautions
- `calculators/` — self-contained place-value, base-conversion, carry-step, borrow-step, zero-role, and positional-score calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, zero notes, transmission cautions, algorism notes, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and interpretive. They are educational scaffolds, not philological editions, numeral palaeography, or exhaustive historical proof.

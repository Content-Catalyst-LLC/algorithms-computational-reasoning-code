# Zero and Place Value Notes

Zero has multiple roles in positional calculation:

- placeholder: marks an empty position;
- structural marker: preserves the layout of powers of the base;
- arithmetic object: participates in operations;
- boundary case: creates procedural limits such as division by zero;
- conceptual bridge: links notation, arithmetic, algebra, and abstraction.

Place value makes digit meaning positional. A digit is not a fixed value by itself; it receives magnitude from its place in the representation.

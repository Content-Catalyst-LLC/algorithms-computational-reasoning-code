# Use Boundaries

This companion folder is educational and interpretive. It is not a complete historical database, not a philological edition, not a manuscript catalogue, not a digit-glyph palaeography resource, and not proof of a single-civilization ownership claim.

Use these files to support:

- teaching;
- conceptual mapping;
- historical framing;
- representation and procedure analysis;
- structured comparison of transmission themes;
- computational scaffolds for humanities-oriented analysis.

Do not use them to erase either Indian origins or Arabic/Islamic-world transmission.

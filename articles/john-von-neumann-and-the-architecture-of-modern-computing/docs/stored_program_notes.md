# Stored-Program Notes

Stored-program computing means that instructions are stored in memory and fetched for execution.

Key implications:

- programs become internal symbolic objects;
- software can be loaded, copied, modified, generated, compiled, and interpreted;
- instruction and data storage become architectural questions;
- control flow can be encoded in memory;
- program-as-data creates both flexibility and security risk.

Stored-program computing is central to modern software.

# von Neumann Bottleneck Notes

The bottleneck arises when compute capacity is constrained by moving instructions and data between memory and processing units.

Modern responses include:

- cache hierarchies;
- prefetching;
- high-bandwidth memory;
- vectorization;
- GPUs and accelerators;
- data locality;
- near-memory computing;
- distributed data placement.

For AI systems, memory bandwidth and data movement can be as important as arithmetic throughput.

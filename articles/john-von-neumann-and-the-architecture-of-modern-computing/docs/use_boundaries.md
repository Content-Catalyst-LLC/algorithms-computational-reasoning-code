# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching stored-program architecture;
- explaining memory, control, arithmetic, input, and output;
- connecting architecture to software layers and AI infrastructure;
- discussing bottlenecks, performance, and data movement;
- framing collaborative credit responsibly;
- supporting architecture-aware governance.

Do not treat synthetic scores as hardware benchmarks, formal architecture proofs, or a replacement for computer-architecture engineering.

# Governance Review

Algorithmic fairness and computational justice governance should include:

- fairness scope and affected-group definition;
- measurement-validity review;
- metric-selection rationale;
- disaggregated performance review;
- calibration and error-rate analysis;
- individual and procedural fairness review;
- contestability and appeal pathways;
- remediation and recurrence prevention;
- deployment monitoring and drift review;
- accountable owners for system change.

A system should not be called fair merely because it passes one metric.

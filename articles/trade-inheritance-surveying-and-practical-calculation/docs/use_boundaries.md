# Use Boundaries

This companion folder is educational and interpretive. It is not legal advice, financial advice, surveying guidance, tax guidance, or an administrative decision tool.

Use these files to support:

- teaching;
- conceptual mapping;
- historical framing;
- procedure analysis;
- structured comparison of practical-calculation themes;
- computational scaffolds for humanities-oriented analysis.

Do not use the calculators for actual legal, commercial, inheritance, surveying, or financial decisions.

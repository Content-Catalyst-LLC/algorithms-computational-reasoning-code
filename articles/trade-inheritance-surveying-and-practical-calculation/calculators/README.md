# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `trade_price_calculator.py` — compute quantity × unit price.
- `currency_exchange_calculator.py` — convert an amount by an exchange rate.
- `inheritance_share_calculator.py` — distribute a total by relative weights.
- `survey_rectangle_area_calculator.py` — compute area from length and width.
- `survey_trapezoid_area_calculator.py` — compute trapezoid area.
- `unit_conversion_calculator.py` — convert quantity by a conversion factor.
- `ledger_balance_checker.py` — check whether debits and credits balance.
- `practical_score_calculator.py` — average procedure, representation, institution, verification, transmission, and modern resonance.

## R calculators

- `practical_score.R` — compute practical calculation score.

# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not legal advice, certification, compliance determination, production approval, model validation, or safety assurance.

## Python calculators

- `future_risk_calculator.py` — estimate future algorithmic risk.
- `readiness_calculator.py` — estimate deployment readiness.
- `automation_risk_calculator.py` — estimate automation risk from stakes, opacity, delegation, and irreversibility.
- `meaningful_review_calculator.py` — estimate whether human review is meaningful.
- `no_go_calculator.py` — evaluate no-go/refusal criteria.
- `computational_literacy_calculator.py` — estimate literacy coverage.
- `ai_agent_boundary_calculator.py` — flag weak agent boundaries.
- `generated_code_governance_calculator.py` — flag AI-generated code governance gaps.

## R calculators

- `future_risk_score.R` — compute future risk score in R.

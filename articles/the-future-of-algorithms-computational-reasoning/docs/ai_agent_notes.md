# AI Agent Notes

AI agents and tool-using systems require stricter boundaries than output-only systems.

Review questions:

- What tools can the agent call?
- What permissions does each tool carry?
- What actions require confirmation?
- What state or memory is retained?
- What logs are kept?
- What errors are reversible?
- What actions are prohibited?
- Who is accountable for the result?

Future agents should be bounded assistants, not unaccountable actors.

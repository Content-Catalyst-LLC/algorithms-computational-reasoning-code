# Agent-Based Calculators

Self-contained calculator scripts for small agent-based threshold-adoption examples.

Python:

```bash
python3 calculators/python/agent_based_calculator.py --agents 100 --threshold 0.5 --steps 30 --seed 7
```

R:

```bash
Rscript calculators/r/model_calculator.R 100 0.5 30 7
```

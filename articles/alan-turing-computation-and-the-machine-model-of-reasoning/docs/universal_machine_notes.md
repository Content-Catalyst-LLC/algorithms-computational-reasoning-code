# Universal Machine Notes

A universal machine can simulate another machine when given a description of that machine and its input.

Important implications:

- instructions can be represented as data;
- one general machine can carry out many procedures;
- program descriptions can be stored, copied, interpreted, and executed;
- software becomes thinkable as symbolic description;
- universality explains general-purpose computation in principle.

Universality should not be confused with unlimited practical capability.

# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching Turing machines and computability;
- explaining effective procedure and universal computation;
- distinguishing computability from feasibility;
- discussing undecidability and halting;
- connecting Turing to AI caution and governance.

Do not treat the synthetic scores as formal proofs or as a replacement for computability theory, mathematical logic, software verification, or specialist history-of-computing scholarship.

# Turing Machine Notes

A Turing machine is an abstract model, not a physical laptop.

Core components:

- tape: symbolic memory medium;
- squares: discrete cells;
- alphabet: allowed symbols;
- head: reads, writes, and moves;
- state: finite internal condition;
- transition rule: maps current state and symbol to next state, symbol, and movement.

The model matters because it gives effective procedure a mathematical form.

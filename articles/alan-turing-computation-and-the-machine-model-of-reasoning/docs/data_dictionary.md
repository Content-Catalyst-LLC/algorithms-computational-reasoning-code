# Data Dictionary

## turing_themes.csv

- `theme_id`: interpretive theme in the Turing machine reasoning map.
- `formalization`: degree to which the theme clarifies effective procedure mathematically.
- `machine_abstraction`: degree to which the theme abstracts computation as machine action.
- `symbolic_representation`: importance of symbols, tape marks, encodings, and descriptions.
- `universality`: relevance to universal machines, simulation, and program-as-data.
- `decidability`: relevance to decision problems, decidability, and undecidability.
- `limit_awareness`: degree to which the theme clarifies formal or practical limits.
- `reasoning_relevance`: degree to which the theme explains formal reasoning.
- `ai_relevance`: degree to which the theme informs machine-intelligence debates.
- `governance_caution`: degree to which the theme supports responsible automation.
- `modern_resonance`: degree to which the theme clarifies modern computation.

## turing_machine_reasoning_map.csv

- `reasoning_score`: average of the theme dimensions.
- `interpretive_status`: core, major, or supporting Turing machine reasoning thread.

## interpretation_cautions.csv

- `caution`: interpretive risk to avoid.
- `meaning`: explanation of the caution.

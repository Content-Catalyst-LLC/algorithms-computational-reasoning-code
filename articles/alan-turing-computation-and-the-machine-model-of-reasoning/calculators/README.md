# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not formal proof tools.

## Python calculators

- `reasoning_score_calculator.py` — average Turing machine reasoning dimensions.
- `turing_transition_simulator.py` — run a tiny transition system over a tape.
- `unary_increment_machine.py` — simple one-symbol increment example.
- `halting_trace_calculator.py` — trace bounded execution to illustrate halting vs non-halting uncertainty.
- `decidability_scope_helper.py` — classify example problem families.
- `universal_description_calculator.py` — represent a toy machine description as data.
- `imitation_game_evaluator.py` — score a behavioral-test framing.
- `ai_overclaim_caution_calculator.py` — flag overclaims about machine reasoning.

## R calculators

- `turing_reasoning_score.R` — compute Turing reasoning score.

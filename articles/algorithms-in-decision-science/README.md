# Algorithms in Decision Science

This companion folder supports the Sustainable Catalyst article **Algorithms in Decision Science**.

It provides reproducible, teaching-oriented workflows for computational decision support, forecasting, thresholds, prioritization, triage, expected value, expected loss, multi-criteria decision analysis, calibration, uncertainty, human review, contestability, feedback effects, and responsible governance.

## Article sequence

- Previous: `when-algorithms-should-not-be-used`
- Current: `algorithms-in-decision-science`
- Next: `algorithms-in-systems-modeling`

## What this folder demonstrates

- how forecasts, thresholds, and expected consequences connect to action;
- how predicted probabilities should be interpreted with uncertainty and calibration;
- how expected value and expected loss can support but not replace judgment;
- how decision-support readiness can combine calibration, uncertainty communication, human review, contestability, and governance;
- how thresholds, loss functions, and decision rules should be documented as governance artifacts;
- how calculators, tests, SQL queries, Canvas cards, and multilingual examples can support accountable algorithmic decision science.

## Repository structure

- `python/` — reference decision-support audit workflow and tests
- `r/` — diagnostic summaries and figures
- `sql/` — schema and queries for decisions, thresholds, expected value, expected loss, readiness, and governance
- `calculators/` — self-contained expected-value, expected-loss, threshold, readiness, and multi-criteria calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, threshold notes, governance review, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and educational. They are not legal, compliance, audit certification, civil-rights, medical, credit, employment, public-benefits, or operational advice.

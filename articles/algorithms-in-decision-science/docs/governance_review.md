# Governance Review

Algorithmic decision support governance should include:

- decision statement and delegated authority review;
- forecast documentation and calibration evidence;
- threshold rationale;
- expected value and expected loss review;
- multi-criteria tradeoff review;
- human review protocol;
- contestability and remediation;
- monitoring for reliance, drift, and feedback;
- stop, rollback, and retirement authority.

Prediction does not justify action unless the decision process is governed.

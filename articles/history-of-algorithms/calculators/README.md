# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not historical proof tools.

## Python calculators

- `history_score_calculator.py` — average algorithm-history dimensions.
- `timeline_layer_classifier.py` — classify rough historical layer from mechanization/formalization/programmability/governance scores.
- `procedure_feature_score.py` — score input, steps, conditions, iteration, termination, and verification.
- `complexity_growth_calculator.py` — compare simple n, n log n, n^2, and 2^n growth.
- `computability_status_helper.py` — label example problem families as computable, undecidable, or governance-dependent.
- `governance_relevance_calculator.py` — score audit, contestability, harm, transparency, and human judgment.
- `algorithm_ancestor_caution_calculator.py` — flag modern-code projection into older procedure.

## R calculators

- `history_score.R` — compute algorithm history score.

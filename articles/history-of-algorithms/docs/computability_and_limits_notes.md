# Computability and Limits Notes

Modern algorithm history includes formal limits.

Important ideas:

- effective procedure;
- lambda calculus;
- Turing machine;
- decidability;
- undecidability;
- computable function;
- universal machine;
- halting problem;
- complexity;
- tractability.

The modern algorithm is not only a practical method; it is also a formal object whose powers and limits can be studied.

# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching algorithm history;
- mapping procedural traditions;
- distinguishing modern algorithms from algorithm-like ancestors;
- comparing representation, proof, mechanization, formalization, and governance;
- building public-facing historical explanations.

Do not treat the synthetic scores as historical facts or as a replacement for specialist historical, philological, mathematical, computer-science, or technical scholarship.

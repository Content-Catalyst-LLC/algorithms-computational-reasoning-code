# Responsible Use Notes

This repository uses synthetic educational data. It should not be used as a real decision system.

Classification can affect access, visibility, safety, eligibility, reputation, burden, or opportunity. Any real system should document:

- decision purpose;
- category definitions;
- feature sources;
- threshold rationale;
- score interpretation and calibration;
- false-positive and false-negative consequences;
- fairness and distributional effects;
- human review paths;
- appeal or contestability process;
- model and rule versioning;
- audit logs;
- governance responsibilities.

A threshold is not just a number. It is a boundary with consequences.

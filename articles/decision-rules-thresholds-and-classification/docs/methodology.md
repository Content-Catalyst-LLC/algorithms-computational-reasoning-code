# Methodology

This article folder treats decision rules, thresholds, and classification as accountable boundary-setting processes.

The workflows demonstrate:

1. how threshold rules turn scores into labels;
2. how confusion-matrix metrics change as thresholds move;
3. how false positives and false negatives carry different costs;
4. how rule documentation, feature documentation, calibration review, human review, appeals, fairness review, traceability, and governance can be scored as audit dimensions;
5. how synthetic examples can support explanation without using sensitive or real personal data.

The examples are intentionally small so that users can inspect every calculation.

# Calculator Layer: Model Validation, Testing, and Computational Evidence

This calculator layer provides small, self-contained tools for model validation review:

- RMSE
- MAE
- bias
- calibration slope/intercept
- classification threshold metrics
- validation evidence checklist scoring

Run:

```bash
python3 calculators/python/model_calculator.py
Rscript calculators/r/model_calculator.R
bash calculators/run_calculator_smoke_tests.sh
```

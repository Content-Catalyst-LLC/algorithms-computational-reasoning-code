# Lambda-Calculus Governance Checklist

- Parse expressions into variables, abstractions, and applications.
- Identify free and bound variables.
- Use alpha conversion to avoid variable capture.
- Record substitution steps explicitly.
- State the evaluation strategy.
- Preserve reduction traces.
- Identify nontermination and recursion risks.
- Add type discipline when guarantees matter.
- Connect formal examples to implementation with care.
- Document what the formal expression represents and what it omits.

# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching algorithm historiography;
- evaluating origin-story claims;
- separating word, concept, practice, transmission, and formalization layers;
- identifying anachronism and overclaiming;
- revising public-facing histories of algorithmic reasoning.

Do not treat the synthetic scores as historical facts or as a replacement for specialist historical, philological, codicological, or technical scholarship.

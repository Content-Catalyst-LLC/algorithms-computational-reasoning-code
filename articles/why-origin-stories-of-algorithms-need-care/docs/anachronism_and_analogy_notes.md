# Anachronism and Analogy Notes

Useful analogies:

- procedure as algorithm-like;
- table as data-structure-like;
- translation as porting-like;
- mechanical automaton as control-system-like;
- algebraic rule as program-like.

Required caution:

- analogy is not identity;
- premodern procedures are not modern software;
- automata are not modern robots;
- tables are not digital databases;
- verbal algebra is not modern symbolic algebra;
- algorithm-like is not the same as formal computability theory.

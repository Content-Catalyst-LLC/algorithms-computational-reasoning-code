# Method Notes

This folder treats algorithm origin stories as historiographic objects.

The core method is:

1. Identify the claim: word origin, concept origin, practice origin, institution, or modern definition.
2. Ask what evidence supports the claim: text, manuscript, translation, usage, commentary, or institution.
3. Check whether the claim overstates invention, priority, direct influence, or continuity.
4. Identify earlier, parallel, and later traditions that complicate the story.
5. Separate etymology from conceptual and procedural development.
6. Mark modern analogies as analogies rather than literal identities.
7. Include transmission actors: translators, scribes, teachers, users, institutions, and tools.
8. Rewrite the story using precise verbs: preserved, translated, systematized, popularized, formalized, mechanized.

The goal is to make origin stories useful without making them misleading.

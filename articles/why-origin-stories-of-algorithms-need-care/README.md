# Why Origin Stories of Algorithms Need Care

This companion folder supports the Sustainable Catalyst article **Why Origin Stories of Algorithms Need Care**.

It provides reproducible, teaching-oriented workflows for evaluating algorithm origin stories with care: word history, concept history, practice history, transmission history, modern formalization, etymology, al-Khwārizmī, algorism, algebra, older procedural traditions, translation networks, anachronism, great-man narratives, civilizational erasure, token inclusion, and AI-era public history.

## What this folder demonstrates

- how to separate word origin from concept origin, procedural practice, institutional reception, and modern formalization;
- how to honor al-Khwārizmī without turning his role into a myth;
- how algorism and algebra belong to algorithm history without exhausting it;
- how older procedures can be algorithm-like without being modern programs or formal algorithms;
- how translation, reception, copying, teaching, commentary, and institutional memory make methods durable;
- how great-man stories can be revised into network histories;
- how to avoid both civilizational erasure and token inclusion;
- how AI-era algorithm stories need stronger historiographic caution.

## Repository structure

- `python/` — reference origin-story care map workflow and tests
- `r/` — historiography diagnostics and figures
- `sql/` — schema and queries for origin-story themes, scores, and cautions
- `calculators/` — self-contained layer-confusion, etymology-risk, anachronism, network-credit, evidence-scope, and origin-care calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, historiography notes, anachronism notes, al-Khwārizmī notes, origin-story rewrite guide, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and interpretive. They are educational scaffolds for historical and conceptual analysis, not claims about a single origin, lineage, or authoritative chronology.

# Algorithms in Education and Learning Systems

This companion folder supports the Sustainable Catalyst article **Algorithms in Education and Learning Systems**.

It provides reproducible, teaching-oriented workflows for adaptive learning, learning analytics, student support, assessment, automated scoring, advising, placement, curriculum recommendation, accessibility, student data privacy, educational equity, teacher judgment, generative tutoring, institutional governance, audit trails, and responsible learning-system design.

## Article sequence

- Previous: `algorithms-in-climate-energy-and-infrastructure`
- Current: `algorithms-in-education-and-learning-systems`
- Next: `algorithms-in-labor-management-and-organizational-systems`

## What this folder demonstrates

- how education and learning systems can be reviewed for learner impact, instructional impact, equity readiness, privacy readiness, pedagogical validity, human review, accessibility readiness, monitoring, and governance readiness;
- how mastery probability, alert thresholds, recommendation scores, learning gain, governance readiness, and learning-system risk can be calculated in small examples;
- how high-impact student systems can require student-impact review;
- how automated scoring or weak instructional systems can require pedagogical-validity review;
- how education systems with weak equity safeguards can require educational-equity review;
- how student data systems can require privacy review;
- how learning-system governance requires pedagogical review, equity audit, accessibility review, student privacy safeguards, teacher override, contestability, monitoring, audit trails, and stop rules.

## Repository structure

- `python/` — reference learning-system governance audit workflow and tests
- `r/` — diagnostic summaries and figures
- `sql/` — schema and queries for learning systems, learning risk scores, and governance controls
- `calculators/` — self-contained mastery, alert-threshold, recommendation-score, learning-gain, governance-readiness, and learning-risk calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, pedagogical review, privacy review, equity review, accessibility review, governance review, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and educational. They are not educational policy advice, legal advice, privacy advice, assessment validation certification, accessibility certification, admissions guidance, or operational guidance.

# Pedagogical Review Notes

Education algorithm review should ask:

- What learning goal does the system support?
- What evidence of learning is used?
- What does the system leave out?
- Does it support transfer, explanation, reflection, curiosity, creativity, or judgment?
- Does it narrow learning toward machine-readable behavior?
- How do teachers interpret and challenge outputs?
- Does feedback help learners improve?
- Does the system preserve student agency?

Prediction is not the same as teaching. Engagement is not the same as learning.

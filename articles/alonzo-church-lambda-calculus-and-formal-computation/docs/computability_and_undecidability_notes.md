# Computability and Undecidability Notes

Church's work contributes to:

- effective calculability;
- lambda definability;
- recursive functions;
- Church's thesis;
- Church's theorem;
- negative solution to the Entscheidungsproblem;
- convergence with Turing computability.

Formal computation is powerful because it defines what can be calculated. It is responsible because it also reveals what cannot be decided by a general method.

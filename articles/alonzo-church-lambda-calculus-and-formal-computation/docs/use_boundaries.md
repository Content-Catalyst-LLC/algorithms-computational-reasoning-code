# Use Boundaries

This companion folder is educational and interpretive.

Use these files to support:

- teaching lambda calculus concepts;
- comparing Church and Turing;
- explaining formal computation, reduction, and encoding;
- discussing computability and undecidability;
- connecting Church to functional programming, types, and proof systems;
- supporting AI-era caution about formal reasoning.

Do not treat synthetic scores as formal proofs or as a replacement for lambda-calculus theory, mathematical logic, programming-language semantics, or specialist history-of-computing scholarship.

# Data Dictionary

## Synthetic measurement records

- `unit_id`: Synthetic record identifier.
- `group`: Synthetic group marker used only for measurement-gap auditing.
- `latent_need`: Hidden teaching construct used to demonstrate construct validity problems.
- `true_construct_positive`: Synthetic construct indicator derived from latent need.
- `prior_access`: Historical access feature; missing values represent patterned institutional visibility.
- `prior_access_missing`: Indicator that prior access is absent.
- `engagement_count`: Behavioral proxy feature.
- `application_completeness`: Documentation proxy feature.
- `delay_days`: Process timing feature.
- `proxy_score`: Thresholded proxy score used to assign observed labels.
- `observed_label`: Imperfect administrative label.
- `label_disagrees_with_construct`: Synthetic disagreement flag.

These fields are educational artifacts. They are not real people, real institutions, or real eligibility records.

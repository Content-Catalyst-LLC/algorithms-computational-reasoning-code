# Entropy Notes

Shannon information for an event:

`I(x) = -log2 p(x)`

Entropy for a discrete source:

`H(X) = -sum_x p(x) log2 p(x)`

Interpretation:

- rare events carry more information than predictable events;
- entropy is expected information from a source;
- entropy depends on the probability model;
- high entropy is not the same as meaning, truth, or importance;
- entropy provides a theoretical compression boundary under model assumptions.

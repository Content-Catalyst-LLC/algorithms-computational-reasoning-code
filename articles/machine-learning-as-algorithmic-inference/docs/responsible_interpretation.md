# Responsible Interpretation

Machine-learning outputs should be interpreted as conditional outputs of a data/model/process pipeline. A score is not a complete explanation. A classification is not an ethical judgment. An average metric is not a guarantee of fair or reliable use.

Responsible interpretation asks:

- What did the model learn from?
- What does the label actually measure?
- What changes when the threshold changes?
- Who bears the cost of each error type?
- Where will the model fail when conditions change?
- Who can challenge a model-supported decision?

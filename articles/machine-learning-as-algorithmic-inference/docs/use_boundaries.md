# Use Boundaries

This companion folder should be used for education and reproducible demonstration only. It should not be treated as a deployable model.

A machine-learning workflow becomes institutionally meaningful only when the following are addressed:

- purpose and scope,
- data provenance,
- feature and label validity,
- evaluation design,
- uncertainty and calibration,
- subgroup error,
- decision thresholds,
- human review,
- appeal pathways,
- monitoring and retirement criteria.

The core article position is that machine learning supports inference, but inference is not the same thing as institutional justification.

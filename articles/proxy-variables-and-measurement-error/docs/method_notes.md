# Method Notes

This folder treats proxy variables as measurement claims.

The core method is:

1. Define the intended construct.
2. Identify the proxy variable and its source.
3. Map the measurement process.
4. Identify random, systematic, differential, missingness, label, and strategic error.
5. Validate the proxy against stronger evidence where possible.
6. Disaggregate validation across relevant groups and contexts.
7. Run sensitivity analysis under plausible error assumptions.
8. Set use boundaries and monitor measurement drift.

The examples are dependency-light and educational.

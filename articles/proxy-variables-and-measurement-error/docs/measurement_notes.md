# Measurement Notes

Data are not direct access to reality.

A proxy variable is an observed stand-in for a construct.
Measurement error is the difference between the observed value and the intended value.
Construct validity asks whether the proxy measures what it claims.
Missingness should be interpreted as information about the measurement system.
Differential error can create fairness risk even when the model architecture appears neutral.
Labels should be treated as evidence produced by processes, not unquestioned truth.

# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `table_lookup_calculator.py` — retrieve a value from a small synthetic table by argument.
- `linear_interpolation_calculator.py` — estimate between two table entries.
- `mean_plus_correction_calculator.py` — compute predicted value from mean and correction.
- `cycle_next_date_calculator.py` — compute a next cycle marker from date, cycle length, and adjustment.
- `calendar_offset_calculator.py` — compute days between two synthetic calendar markers.
- `prediction_error_calculator.py` — compare predicted and observed values.
- `prediction_score_calculator.py` — average table structure, procedure, predictive function, institution, correction, transmission, and modern resonance.

## R calculators

- `prediction_score.R` — compute astronomical prediction score.

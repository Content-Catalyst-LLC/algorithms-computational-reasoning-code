# Use Boundaries

This companion folder is educational and interpretive. It is not astronomical, calendrical, religious, navigational, legal, agricultural, or administrative advice.

Use these files to support:

- teaching;
- conceptual mapping;
- historical framing;
- table-procedure analysis;
- structured comparison of astronomical prediction themes;
- computational scaffolds for humanities-oriented analysis.

Do not use the calculators for real astronomical, calendar, navigation, ritual, or legal decisions.

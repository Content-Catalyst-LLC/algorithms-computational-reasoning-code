# Calendar Rule System Notes

Calendars are rule-governed systems for organizing time.

Questions to ask:

- What begins the month?
- What counts as the year?
- How are cycles reconciled?
- How are dates converted?
- How are festivals, rituals, seasons, and administration coordinated?
- What role does observation play?
- What role do tables or rules play?

Calendars are computational because they transform cycles into public time.

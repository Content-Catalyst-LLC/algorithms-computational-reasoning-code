# Zīj and Tabular Astronomy Notes

A zīj is an astronomical handbook that typically includes tables and instructions for astronomical calculation.

Important interpretive points:

- A table is stored computation, not a passive list.
- A rule of use is required to move from input to prediction.
- Parameters configure the model.
- Copying and correction are part of computational maintenance.
- Regional adaptation can localize tables for place, calendar, or practice.
- Instruction and teaching make table use reliable.

A zīj can be read as a premodern scientific-computing artifact in manuscript form.

# Transmission Cautions

Avoid these claims:

- Verbal mathematics is automatically vague.
- Medieval procedure is just modern code written in words.
- Symbols are the only form of precision.
- Worked examples are secondary decoration.
- Transmission is passive copying.
- One culture owns the idea of procedure.

Prefer these claims:

- Verbal procedure can be precise inside a trained mathematical tradition.
- Pre-modern procedures can be algorithmic without being programs.
- Symbols and prose are different media for organizing operations.
- Examples, diagrams, tables, manuscripts, teachers, and translations all help procedures survive.

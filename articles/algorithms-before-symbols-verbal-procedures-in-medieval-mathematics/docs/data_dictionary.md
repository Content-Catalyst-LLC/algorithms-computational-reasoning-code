# Data Dictionary

## verbal_procedure_themes.csv

- `theme_id`: interpretive theme in algorithms before symbols.
- `procedural_clarity`: degree to which the theme defines repeatable operations.
- `representation_dependence`: importance of prose, examples, diagrams, tables, or named quantities.
- `pedagogical_value`: importance for teaching, exercises, worked examples, or training.
- `transmission_importance`: importance in copying, translation, commentary, or institutional movement.
- `practical_use`: degree of use in commerce, inheritance, astronomy, surveying, administration, or education.
- `modern_resonance`: degree to which the theme clarifies modern computational reasoning.

## verbal_procedure_map.csv

- `verbal_procedure_score`: average of the theme dimensions.
- `interpretive_status`: core, major, or supporting verbal-procedure thread.

## interpretation_cautions.csv

- `caution`: interpretive risk to avoid.
- `meaning`: explanation of the caution.

## verbal_procedure_summary.csv

- `themes_reviewed`: number of verbal procedure themes reviewed.
- `core_threads`: themes marked as core verbal procedure threads.
- `major_threads`: themes marked as major verbal procedure threads.
- `supporting_threads`: themes marked as supporting verbal procedure threads.
- `mean_verbal_procedure_score`: mean interpretive verbal procedure score.

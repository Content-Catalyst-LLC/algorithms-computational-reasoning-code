# Use Boundaries

This companion folder is educational and interpretive. It is not a complete historical database, not a manuscript edition, not a philological catalogue, and not proof of a single-origin claim.

Use these files to support:

- teaching;
- conceptual mapping;
- historical framing;
- verbal procedure analysis;
- structured comparison of procedural themes;
- computational scaffolds for humanities-oriented analysis.

Do not use them to claim that medieval verbal procedures were modern programs or that verbal mathematics was inherently vague.

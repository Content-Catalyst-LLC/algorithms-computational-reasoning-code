# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `instruction_sequence_counter.py` — count ordered verbal instruction steps.
- `problem_classification_calculator.py` — classify a problem type from a controlled vocabulary.
- `worked_example_trace_calculator.py` — trace input, intermediate state, output, and check for a toy procedure.
- `table_interpolation_calculator.py` — linear interpolation between two tabular values.
- `symbolic_compression_calculator.py` — estimate compression from verbal instruction length to symbolic shorthand length.
- `verbal_procedure_score_calculator.py` — average procedural clarity, representation, pedagogy, transmission, practical use, and modern resonance.

## R calculators

- `verbal_procedure_score.R` — compute a verbal procedure score.

# Valves, Floats, and Thresholds Notes

Valves and floats create conditional behavior in mechanical systems.

Interpretive analogies:

- valve: conditional gate;
- float: sensor-like element;
- threshold: if-condition;
- chamber: state container;
- siphon: triggered process;
- outlet: output channel.

Use these analogies carefully. They clarify structure but should not erase historical difference.

# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not engineering tools.

## Python calculators

- `threshold_action_calculator.py` — determine whether a level reaches a threshold.
- `flow_time_calculator.py` — estimate time from volume and flow rate.
- `valve_gate_calculator.py` — model a simple open/closed gate.
- `reservoir_volume_calculator.py` — compute a simple rectangular reservoir volume.
- `pressure_difference_calculator.py` — compute a basic hydrostatic pressure difference.
- `state_transition_calculator.py` — update a toy mechanical state from input and constraint.
- `mechanical_score_calculator.py` — average mechanical structure, sequence, control, hidden state, feedback, history, caution, and modern resonance.

## R calculators

- `mechanical_score.R` — compute mechanical procedure score.

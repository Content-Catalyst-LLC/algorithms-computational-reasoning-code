# Randomized Algorithm Governance Checklist

- Define the randomized step, distribution, seed policy, and pseudo-random generator.
- Classify the procedure: Las Vegas, Monte Carlo, estimator, simulation, heuristic, or randomized infrastructure rule.
- Document error probability, uncertainty interval, variance, and sample-size rationale.
- Record seeds, trial counts, sampled records, generator versions, and run manifests.
- Test multiple seeds, tail behavior, rare cases, and distributional sensitivity.
- Define reproducibility requirements and when unpredictability is required instead.
- Report whether the result is exact, expected, approximate, simulated, or probabilistic.
- Review group-level exposure, sampling coverage, fairness, and downstream consequences.

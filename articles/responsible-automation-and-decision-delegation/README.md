# Responsible Automation and Decision Delegation

This companion folder supports the Sustainable Catalyst article **Responsible Automation and Decision Delegation**.

It provides reproducible, teaching-oriented workflows for responsible automation, decision delegation, automation boundaries, assisted decision-making, constrained automation, delayed automation, escalation, reversibility, contestability, remediation, rollback, retirement, governance controls, and accountable human judgment.

## Article sequence

- Previous: `human-in-the-loop-and-human-judgment`
- Current: `responsible-automation-and-decision-delegation`
- Next: `algorithmic-risk-management-and-ai-governance`

## What this folder demonstrates

- how automation should be treated as delegated authority rather than mere efficiency;
- how delegation readiness can combine evidence quality, validation, reversibility, contestability, governance, and human review;
- how automation reliance can reveal when decision support becomes de facto decision-making;
- how delegation risk rises when stakes are high and readiness is weak;
- how governance registers, calculators, tests, SQL queries, Canvas cards, and multilingual examples can support responsible automation decisions.

## Repository structure

- `python/` — reference delegation readiness audit workflow and tests
- `r/` — diagnostic summaries and figures
- `sql/` — schema and queries for delegation readiness, automation reliance, reversibility, risk, and governance
- `calculators/` — self-contained delegation-readiness, delegation-risk, reversibility, and reliance calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, governance review, delegation boundary notes, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and educational. They are not legal, compliance, audit certification, civil-rights, medical, credit, employment, public-benefits, or operational advice.

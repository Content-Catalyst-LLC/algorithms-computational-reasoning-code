# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `delegation_readiness_calculator.py` — average evidence quality, validation, reversibility, contestability, governance, and human review.
- `delegation_risk_calculator.py` — stakes multiplied by weak delegation readiness.
- `automation_reliance_calculator.py` — automated final actions divided by total decisions.
- `reversibility_calculator.py` — average detection, correction, remedy, and recurrence prevention.

## R calculators

- `delegation_score.R` — compute delegation readiness, automation reliance, and delegation risk.

Run smoke tests:

```bash
bash calculators/run_calculator_smoke_tests.sh
```

# Algorithmic Bias, Data, and Institutional History

This companion folder supports the Sustainable Catalyst article **Algorithmic Bias, Data, and Institutional History**.

It provides reproducible, teaching-oriented workflows for algorithmic bias, historical bias, data provenance, institutional history, representation gaps, measurement bias, label bias, proxy bias, feedback reinforcement, fairness review, contestability, remediation, and responsible algorithmic governance.

## Article sequence

- Previous: `algorithmic-fairness-and-computational-justice`
- Current: `algorithmic-bias-data-and-institutional-history`
- Next: `metrics-feedback-and-algorithmic-failure`

## What this folder demonstrates

- how data can encode institutional history;
- how selection gaps, representation gaps, label gaps, proxy risk, and measurement weakness can be reviewed together;
- how historical risk can be modeled as a combination of provenance risk, measurement weakness, proxy risk, and weak remediation;
- how bias audits should connect statistical patterns with institutional origins and corrective obligations;
- how governance artifacts can document provenance, representation, measurement, contestability, remediation, and accountability.

## Repository structure

- `python/` — reference bias, data, and institutional history audit workflow and tests
- `r/` — diagnostic summaries and figures
- `sql/` — schema and governance queries for bias history, representation, labels, proxies, and remediation
- `calculators/` — self-contained representation-gap, label-gap, historical-risk, and selection-gap calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, provenance notes, measurement notes, governance review, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and educational. They are not legal, civil-rights, compliance, employment, credit, public-benefits, medical, policing, education, or operational advice.

# Provenance Notes

Data provenance is a fairness and governance issue.

Ask:

- Which institution produced the data?
- Why were the records collected?
- Which categories were available?
- Who was observed, served, excluded, diagnosed, policed, moderated, rewarded, or ignored?
- Which policy choices shaped the records?
- Were previous algorithmic systems involved?
- Can affected people correct records?
- Are the data appropriate for the new decision?

A dataset is not only a table. It is an institutional history in computational form.

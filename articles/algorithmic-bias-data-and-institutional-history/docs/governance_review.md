# Governance Review

Bias, data, and institutional history governance should include:

- data provenance review;
- institutional-history mapping;
- representation and missingness analysis;
- label-generation review;
- proxy and feature review;
- fairness metric review;
- feedback-loop monitoring;
- contestability and appeal pathways;
- remediation and recurrence prevention;
- named owners for corrective action.

A responsible institution remains accountable for the history its systems learn from.

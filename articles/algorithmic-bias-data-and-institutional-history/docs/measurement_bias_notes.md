# Measurement Bias Notes

Measurement bias appears when a variable does not validly or consistently measure the intended construct.

Common proxy risks:

- cost as proxy for health need;
- arrest as proxy for crime;
- engagement as proxy for value;
- test score as proxy for learning;
- credit history as proxy for reliability;
- prior hiring as proxy for merit;
- attendance as proxy for commitment;
- account reports as proxy for harmful behavior.

Measurement review should ask what the variable actually measures and whether that meaning differs across groups or contexts.

# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not historical proof tools.

## Python calculators

- `imagination_score_calculator.py` — average Lovelace computation-imagination dimensions.
- `bernoulli_number_calculator.py` — compute Bernoulli numbers using a simple Akiyama-Tanigawa method.
- `operation_sequence_calculator.py` — summarize ordered operation sequences.
- `symbolic_generality_calculator.py` — score whether a domain has representation, relation, operation, pattern, and output.
- `machine_limit_calculator.py` — flag machine agency overclaims.
- `authorship_credit_calculator.py` — score translation, notes, interpretation, collaboration, and procedure.
- `lovelace_myth_caution_calculator.py` — flag slogan, dismissal, and projection risks.

## R calculators

- `lovelace_imagination_score.R` — compute Lovelace imagination score.

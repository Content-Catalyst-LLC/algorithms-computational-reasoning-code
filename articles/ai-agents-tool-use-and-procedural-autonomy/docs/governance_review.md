# Governance Review

Agentic AI systems should be governed through:

- intended-use and prohibited-use statements;
- tool registries and permission matrices;
- least-privilege access;
- approval gates for write, external-write, execute, and high-risk actions;
- state and tool-call logs;
- prompt-injection defenses;
- monitoring, rollback, and incident response;
- human accountability and contestability.

The more an agent can act, the stronger the governance layer must be.

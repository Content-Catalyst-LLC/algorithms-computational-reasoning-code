# Method Notes

This folder treats AI agents as governed procedural systems.

The core method is:

1. Define the workflow purpose and stakes.
2. Register every available tool.
3. Classify tools by read, compute, execute, draft, write, or external-write action type.
4. Assign risk scores and approval requirements.
5. Audit planned actions as state-action-observation records.
6. Escalate high-risk or unauthorized actions.
7. Recommend autonomy level from evidence.
8. Preserve logs, governance records, and use-boundary statements.

The examples are dependency-light and educational.

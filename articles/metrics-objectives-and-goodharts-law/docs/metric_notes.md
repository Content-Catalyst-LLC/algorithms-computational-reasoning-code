# Metric Notes

Metrics are partial representations.

A useful metric should be valid, reliable, interpretable, and robust under pressure.
A target should be reviewed for incentive effects.
A proxy should be validated against the true goal.
A dashboard should show limits and uncertainty.
A benchmark should be paired with deployment evidence.
A reward signal should be tested for loopholes and unintended strategies.

Goodhart's Law is not a rejection of measurement. It is a warning that measurement changes behavior.

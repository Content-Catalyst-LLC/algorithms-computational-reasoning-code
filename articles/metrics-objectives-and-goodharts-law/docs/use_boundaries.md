# Use Boundaries

This companion folder is educational. It should not be used as a production metric-governance system without domain validation.

Do not use the synthetic scores in this folder for real institutional ranking, evaluation, incentive design, or automated decision-making.

Before deploying metrics in practice, require:

- domain expert review;
- stakeholder review;
- proxy validation;
- gaming analysis;
- guardrail design;
- monitoring;
- appeal and correction mechanisms;
- documented ownership.

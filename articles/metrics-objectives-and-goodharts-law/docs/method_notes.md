# Method Notes

This folder treats metrics as governance instruments.

The core method is:

1. State the real objective.
2. Define the proxy metric.
3. Validate the proxy-goal relationship.
4. Assess optimization pressure.
5. Identify gaming and reward-hacking pathways.
6. Add guardrails and counter-metrics.
7. Monitor drift and feedback loops.
8. Review, revise, or retire weak metrics.

The examples are dependency-light and educational.

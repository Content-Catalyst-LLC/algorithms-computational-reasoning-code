# Al-Jabr wa'l-Muqabalah: Algebra as Rule-Governed Problem Solving

This companion folder supports the Sustainable Catalyst article **Al-Jabr wa'l-Muqabalah: Algebra as Rule-Governed Problem Solving**.

It provides reproducible, teaching-oriented workflows for mapping algebra as rule-governed problem solving: al-jabr, al-muqabalah, restoration, balancing, equation-type classification, unknown quantities, verbal algebra, completing the square, geometric demonstration, practical word problems, algebraic transformation, transmission, and origin-story cautions.

## What this folder demonstrates

- how al-jabr and al-muqabalah can be modeled as procedural operations rather than merely as word origins;
- how early algebra can be studied through classification, transformation, representation, demonstration, practical use, transmission, and modern computational resonance;
- how algebraic procedure preserves relations while transforming problem states;
- how completing the square, balancing, cancellation, equation classification, and unknown representation can be implemented as small calculators;
- how careful interpretation avoids projecting modern symbolic notation backward while still recognizing algorithmic structure.

## Repository structure

- `python/` — reference rule-governed algebraic procedure map workflow and tests
- `r/` — algebraic procedure diagnostics and figures
- `sql/` — schema and queries for procedure themes, scores, and interpretation cautions
- `calculators/` — self-contained balancing, cancellation, equation-type, completing-square, unknown-check, and procedure-score calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, restoration/balancing notes, completing-square notes, origin-story cautions, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and interpretive. They are educational scaffolds, not philological editions or exhaustive historical proof.

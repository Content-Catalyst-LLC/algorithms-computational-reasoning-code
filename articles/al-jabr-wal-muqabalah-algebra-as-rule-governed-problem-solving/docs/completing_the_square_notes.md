# Completing the Square Notes

Completing the square is a procedural transformation.

The solver:

1. Recognizes a quadratic form.
2. Isolates the relevant terms.
3. Adds a completing term.
4. Creates a perfect-square structure.
5. Reduces the problem to a square-root step.
6. Solves and checks the result.

The historical importance is not only the result. It is the transformation of an unsolved relation into a recognizable form.

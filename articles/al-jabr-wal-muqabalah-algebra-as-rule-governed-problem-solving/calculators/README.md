# Calculators

These small calculators support article concepts without external packages.

## Python calculators

- `balance_equation_calculator.py` — add the same quantity to both sides.
- `cancel_common_term_calculator.py` — remove the same common term from both sides.
- `equation_type_classifier.py` — classify a toy polynomial degree.
- `complete_square_calculator.py` — compute completing term `(b/2)^2`.
- `unknown_check_calculator.py` — check whether a candidate solution satisfies `a*x + b = c`.
- `algebraic_procedure_score_calculator.py` — average interpretive dimensions.

## R calculators

- `algebraic_procedure_score.R` — compute a rule-governed algebraic procedure score.

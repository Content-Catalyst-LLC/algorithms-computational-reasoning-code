# Methodology

The ranking audit decomposes a visibility system into candidate generation, filtering, scoring, sorting, recommendation, traceability, and governance. It does not claim to model production recommender systems. It provides educational scaffolding for asking whether a ranking system can explain what entered the candidate set, what was filtered, which signals mattered, how candidates were scored, how alternatives were treated, whether diversity and fairness were reviewed, and how feedback loops are controlled.

The synthetic scores are normalized between 0 and 1. The audit score is a weighted composite. The risk score is the average weakness across governance dimensions.

# Calculators

These small calculators support article concepts without external packages. They are educational conceptual scaffolds, not control-system certification tools.

## Python calculators

- `cybernetic_score_calculator.py` — average Wiener cybernetic-feedback dimensions.
- `negative_feedback_calculator.py` — simulate simple movement toward a target.
- `positive_feedback_calculator.py` — simulate amplification over time.
- `control_error_calculator.py` — compute target-observed error and corrective action.
- `delay_effect_helper.py` — compare delayed vs current observations.
- `platform_feedback_loop_calculator.py` — estimate simple amplification from engagement feedback.
- `rl_reward_caution_calculator.py` — flag reward-is-value overclaims.
- `feedback_governance_caution_calculator.py` — flag feedback-governance risks.

## R calculators

- `feedback_score.R` — compute cybernetic feedback score.

# Norbert Wiener, Cybernetics, and Feedback in Computation

This companion folder supports the Sustainable Catalyst article **Norbert Wiener, Cybernetics, and Feedback in Computation**.

It provides reproducible, teaching-oriented workflows for studying Wiener's cybernetic legacy: feedback, control, communication, error signals, prediction, servomechanisms, negative feedback, positive feedback, stability, amplification, homeostasis, self-regulation, automation ethics, human-machine systems, platform loops, reinforcement learning, AI governance, and responsible computational feedback analysis.

## What this folder demonstrates

- how cybernetics studies control and communication in animals and machines;
- how feedback turns computation into a loop of observation, comparison, action, and return;
- how negative feedback can stabilize systems;
- how positive feedback can amplify signals, errors, bias, attention, and institutional effects;
- how prediction, delay, and noise shape control;
- how software systems, platforms, reinforcement learning, and AI systems operate through feedback loops;
- how automation changes human agency, responsibility, and governance;
- why responsible AI requires governing loops, not only models.

## Repository structure

- `python/` — reference Wiener cybernetic feedback map workflow and tests
- `r/` — feedback diagnostics and figures
- `sql/` — schema and queries for cybernetic-feedback themes, scores, and cautions
- `calculators/` — self-contained feedback-score, negative-feedback, positive-feedback, control-error, delay, amplification, platform-loop, and governance-caution calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, cybernetics notes, feedback notes, automation ethics notes, AI governance notes, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and educational. They are not replacements for control theory, cybernetics scholarship, systems engineering, AI governance review, or institutional risk analysis.

# Cybernetics Notes

Cybernetics studies control and communication in animals and machines.

Core structure:

- a system has a state;
- a target or goal defines desired state;
- sensors observe the system or environment;
- a comparator evaluates error;
- a controller selects action;
- the action changes the system;
- the next observation completes the loop.

Cybernetics becomes dangerous when human beings are reduced to components in machine-like control systems.

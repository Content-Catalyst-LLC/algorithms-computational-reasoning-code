# Use Boundaries

This companion folder is educational. It should not be used for real procurement, certification, safety approval, or operational evaluation without domain-specific validation.

Do not make consequential decisions from the synthetic scores in this folder.

Before using an AI system in practice, require:

- task-specific evaluation;
- domain expert review;
- representative test data;
- calibration and uncertainty reporting;
- safety and security testing;
- disaggregated analysis;
- deployment monitoring;
- documented accountability.

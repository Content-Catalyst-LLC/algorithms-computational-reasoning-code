# Metric Notes

Metrics are partial indicators.

Accuracy can hide error severity, group differences, and class imbalance.
Precision and recall highlight different failure modes.
F1 summarizes precision-recall balance but does not capture calibration or cost.
Human preference can reward fluency, confidence, or style rather than truth.
Leaderboards can compress many dimensions into one ranking.
Benchmark scores should be interpreted with uncertainty, task fit, safety, and deployment context.

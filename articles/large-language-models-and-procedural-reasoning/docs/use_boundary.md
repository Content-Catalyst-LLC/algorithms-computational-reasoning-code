# Use Boundary

This repository is for education, reproducible demonstration, and article support.

It should not be used as:

- a production LLM evaluation platform;
- a legal, medical, financial, hiring, or public-service decision tool;
- a guarantee that an LLM output is truthful;
- a replacement for domain expertise or human review;
- a substitute for formal safety, privacy, security, or regulatory assessment.

The workflows use synthetic examples so that reasoning artifacts remain visible and reproducible.

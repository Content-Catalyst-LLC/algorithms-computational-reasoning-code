# Evolutionary Search Governance Checklist

- Define the candidate representation and feasible solution space.
- Define the fitness function, objectives, penalties, and constraints.
- Document mutation, crossover, recombination, repair, replacement, and selection operators.
- Record population size, generation count, mutation rate, crossover rate, selection pressure, seeds, and stopping rules.
- Track best fitness, average fitness, diversity, convergence, and population history.
- Compare against baselines, exact small instances, heuristic alternatives, and held-out tests.
- Run multi-seed and parameter sensitivity analysis.
- Test for reward hacking, benchmark overfitting, safety failures, and edge-case brittleness.
- Preserve run manifests, lineage records, best candidates, and validation results.
- Assign governance responsibility before deployment.

# Use Boundaries

This companion folder is educational and interpretive. It is not operational security advice, penetration-testing guidance, intrusion guidance, or a tool for unauthorized access.

Use these files to support:

- teaching classical cryptanalysis historically;
- conceptual mapping;
- frequency counting demonstrations;
- structured comparison of cryptanalysis themes;
- computational scaffolds for humanities-oriented analysis.

Do not use the calculators for breaking real messages, evading security, attacking systems, or accessing information without authorization.

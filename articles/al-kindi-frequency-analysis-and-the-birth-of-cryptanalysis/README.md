# Al-Kindī, Frequency Analysis, and the Birth of Cryptanalysis

This companion folder supports the Sustainable Catalyst article **Al-Kindī, Frequency Analysis, and the Birth of Cryptanalysis**.

It provides reproducible, teaching-oriented workflows for mapping al-Kindī’s frequency-analysis cryptanalysis as early algorithmic inference over language: letter counting, relative frequency, symbol ranking, substitution hypotheses, pattern testing, linguistic regularity, revision, countermeasures, and ethical boundaries.

## What this folder demonstrates

- how a hidden message can be treated as observable data;
- how letter counting and frequency ranking turn language into evidence;
- how simple substitution ciphers preserve statistical patterns;
- how cryptanalysis uses hypotheses, testing, conflict detection, and revision;
- how al-Kindī’s method should be framed as documented frequency-analysis cryptanalysis, not the invention of all secrecy;
- how cryptanalytic reasoning connects language, mathematics, textual culture, and algorithmic inference;
- how classical cipher examples can be used safely for historical education without presenting modern intrusion techniques.

## Repository structure

- `python/` — reference frequency-analysis and cryptanalysis map workflow and tests
- `r/` — cryptanalysis diagnostics and figures
- `sql/` — schema and queries for cryptanalysis themes, scores, and interpretation cautions
- `calculators/` — self-contained frequency-table, relative-frequency, Caesar demo, substitution demo, hypothesis-score, sample-size, and cryptanalysis-score calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, frequency analysis notes, substitution cipher notes, ethics and boundaries, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and interpretive. They are educational scaffolds for classical cryptanalysis and historical reasoning, not guidance for unauthorized access, modern intrusion, or real-world security attacks.

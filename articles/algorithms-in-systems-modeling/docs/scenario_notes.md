# Scenario Notes

Scenario modeling compares possible futures under different assumptions, interventions, shocks, or policies.

A scenario archive should record:

- scenario name and purpose;
- baseline assumptions;
- intervention assumptions;
- parameter values;
- time horizon;
- external shocks;
- system boundaries;
- outputs reviewed;
- uncertainty ranges;
- sensitivity results;
- affected groups or places;
- decision relevance.

Scenarios are not predictions. They are structured ways of testing how assumptions and interventions shape possible outcomes.

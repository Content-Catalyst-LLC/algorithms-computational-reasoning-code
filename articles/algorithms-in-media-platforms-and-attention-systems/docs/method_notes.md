# Method Notes

This folder treats media-platform algorithms as attention-governance infrastructure.

The core method is:

1. Inventory ranking, recommendation, moderation, monetization, notification, and search systems.
2. Define objectives, metrics, tradeoffs, public-value goals, and prohibited incentives.
3. Map effects on users, creators, communities, public knowledge, and vulnerable groups.
4. Evaluate transparency, contestability, appeals, remedy, and user-control mechanisms.
5. Audit moderation thresholds, demonetization, downranking, and human-review workflows.
6. Monitor virality, manipulation, metric gaming, creator impacts, and feedback loops.
7. Define rollback, pause, friction, and redesign authority for harmful dynamics.

The examples are dependency-light and educational.

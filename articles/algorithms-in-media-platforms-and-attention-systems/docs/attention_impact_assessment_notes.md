# Attention Impact Assessment Notes

An attention impact assessment should record:

- system purpose and owner;
- ranking, recommendation, moderation, monetization, or notification role;
- objective function and key metrics;
- engagement-pressure indicators;
- creator and publisher impacts;
- public-knowledge impacts;
- user agency and control mechanisms;
- moderation and demonetization effects;
- vulnerable-user and community impacts;
- transparency and appeal mechanisms;
- monitoring, incident response, and stop rules.

An attention system should not be evaluated only by engagement growth.

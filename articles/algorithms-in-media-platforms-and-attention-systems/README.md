# Algorithms in Media Platforms and Attention Systems

This companion folder supports the Sustainable Catalyst article **Algorithms in Media Platforms and Attention Systems**.

It provides reproducible, teaching-oriented workflows for feed ranking, recommendation, engagement optimization, creator incentives, advertising auctions, moderation systems, virality, network effects, personalization, user control, transparency, contestability, platform governance, monitoring, and responsible attention-system design.

## Article sequence

- Previous: `algorithms-in-public-policy-and-governance`
- Current: `algorithms-in-media-platforms-and-attention-systems`
- Next: `algorithms-in-finance-markets-and-risk`

## What this folder demonstrates

- how media-platform systems can be reviewed for engagement pressure, transparency, contestability, moderation readiness, creator impact, public-knowledge impact, user control, governance, and monitoring;
- how attention risk and platform risk can be computed from synthetic review indicators;
- how high-risk attention systems can require redesign, public-interest review, stronger guardrails, or governance review;
- how attention-system governance requires inventories, objective review, impact assessment, appeals, moderation governance, monetization review, monitoring, and stop rules;
- how calculators, tests, SQL queries, Canvas cards, and multilingual examples can support responsible platform governance.

## Repository structure

- `python/` — reference attention-system audit workflow and tests
- `r/` — diagnostic summaries and figures
- `sql/` — schema and queries for platform systems, attention risk, and governance controls
- `calculators/` — self-contained ranking, engagement, attention-risk, governance-readiness, platform-risk, and moderation-threshold calculators
- `canvas/` — Canvas-ready cards, schemas, and exports
- `docs/` — method notes, data dictionary, attention-impact notes, moderation notes, governance review, and use boundaries
- language folders — compact examples across Python, R, Julia, SQL, Haskell, Rust, Go, C, C++, Fortran, Java, TypeScript, Prolog, and Racket

All datasets and examples are synthetic and educational. They are not platform compliance advice, legal advice, content moderation advice, public policy advice, advertising advice, safety certification, or operational guidance.
